# Sponge and Interview — 내 삶을 돕는 OS 인터뷰 스킬

스폰지클럽 2기 1주차에서 쓰는 인터뷰 스킬이에요.
가벼운 5스텝 인터뷰를 따라가면, **내 삶을 돕는 OS가 무엇이어야 할지**(일·삶 전반)가 정리된 🟡 *OS 재료 카드* 가 나와요.

## 어떻게 쓰나요

이 레포를 클론(또는 pull)한 상태에서, 클로드 코드에 이렇게 말하면 돼요:

```
나를 인터뷰해줘
```
또는
```
내 OS 재료 찾고 싶어 / 1주차 과제 인터뷰 시작해줘
```

`.claude/skills/` 안에 있어서 클로드 코드가 자동으로 인식해요. 따로 설정할 게 없어요.

## 파일

- `SKILL.md` — 스킬 본체 (인터뷰 5스텝 + 카드 + 과제 연결)
- `시뮬레이션_도토리.md` — 가상 멤버가 처음부터 끝까지 인터뷰를 통과하는 예시. "이렇게 흘러가는구나" 감 잡기용

## 결과물

인터뷰가 끝나면 이런 카드가 나와요:

```
🟡 내 삶을 돕는 OS · 재료 카드
· 영역 / 걸리는 지점 / 지금은 / OS가 된다면 / 한 문장 / 첫 한 걸음
```

이 카드가 곧 1주차 과제 2(내 OS 기획)의 설계도가 돼요.
