# 🧰 skills — 스킬러스 검증 스킬 저장소

검증된 클로드코드 스킬을 모으는 곳이에요. **설치 = 복붙** — 남이 실제로 검증·디벨롭한 스킬을 내 에이전트로 가져오는 거예요.

> 슬랙 = 발견·흐름·후기 / 여기(repo) = 검증·설치 가능한 스킬 자산.

## 누가 올리나요
- **선발대**: 매주 추천 스킬을 디벨롭해서 "내 스킬"로 만들어 여기에 올려요. (핵심 활동)
- **일반크루**: 의무 아니에요. 슬랙에서 구경하다가 ① 마음에 들면 복붙해서 써보고 ② 검증한 스킬이 있으면 자유롭게 올려도 돼요.

## 설치하는 법 (복붙)
1. 쓰고 싶은 `skills/{스킬이름}/SKILL.md` 폴더를 통째로 복사
2. 내 `~/.claude/skills/{스킬이름}/` 에 붙여넣기
3. 끝! 클로드코드에서 바로 쓰여요. (써보고 `reports.md`에 후기·개선안 남겨주면 모두에게 도움 ✨)

## 올리는 법
1. `_template/` 을 복사해서 `skills/{내 스킬이름}/` 으로
2. `SKILL.md`(본체)·`README.md`(설명)·`reports.md`(후기) 채우기
3. 과제 올릴 때처럼 git pull → commit → push

## 폴더 구조
```
skills/
├─ README.md        ← 지금 이 파일
├─ _template/       ← 새 스킬 올릴 때 복사용 양식
└─ {스킬이름}/
   ├─ SKILL.md      ← 스킬 본체 (복붙 설치용)
   ├─ README.md     ← 뭘 해주나·설치법·예시
   └─ reports.md    ← 써본 사람 후기·개선안
```
