# ad-creative-check

캐러셀·카드뉴스·광고 소재·상세페이지 초안을 **"팔리는 관점" 5관문**(후킹 → 타깃 언어 → 흐름 → 전환 → 신뢰)으로 진단해주는 스킬이에요. 콘텐츠를 만들어주는 스킬이 아니라, **만든 콘텐츠의 새는 곳을 찾아주는** 스킬입니다.

우리 클럽에 캐러셀·카드뉴스 "제작" OS는 이미 많죠 (그레이·포비·최강훈 님 것 강추). 이 스킬은 그 뒤에 붙이는 검수 단계예요 — 만들고 → 진단하고 → 1가지 고치고 → 게시.

## 설치
1. 이 폴더(`skills/ad-creative-check/`)를 통째로 복사
2. 내 `~/.claude/skills/ad-creative-check/` 에 붙여넣기
3. 끝!

## 사용 예시
- 캐러셀 이미지(또는 장별 텍스트)를 붙여넣고: `"이 캐러셀 진단해줘"`
- 릴스 대본 초안을 주고: `"게시 전에 봐줘"`
- 반응 없던 콘텐츠를 주고: `"이거 왜 안 터졌는지 봐줘"`

## 결과로 받는 것
- 5관문 각각의 ○/△/✕ 판정
- **가장 크게 새는 곳 1개** + 왜 새는지
- 바로 적용할 수 있는 **대체 문구가 포함된** 수정안 1가지
- 바꾸지 말아야 할 강점

## 만든 사람
잭(유재현, 5조) — 여성의류 쇼핑몰 광고 운영·코칭을 하며 대표님들 콘텐츠를 진단할 때 매번 반복하던 기준을 스킬로 캡슐화했어요. 2주차 과제 "쇼핑몰 진단 OS"의 콘텐츠 버전이라고 보면 됩니다. 써보고 `reports.md`에 후기 남겨주세요!
