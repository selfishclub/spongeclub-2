# ad-creative-check — 소재 진단 스킬

콘텐츠를 예쁘게 만드는 도구는 많다. 이 스킬은 반대쪽 — **만든 콘텐츠가 실제로 반응과 전환을 만들 수 있는지**를 광고 운영자의 눈으로 진단한다. 칭찬하지 말고 새는 곳을 찾아라.

## 언제 쓰나
- 캐러셀·카드뉴스·광고 이미지·릴스 대본·상세페이지 초안을 보여주며 피드백을 요청할 때
- 게시/집행 전 "이대로 나가도 되나?" 최종 점검할 때
- 반응이 안 나온 콘텐츠의 원인을 찾을 때

## 어떻게 동작하나

콘텐츠를 받으면 아래 5개 관문을 **순서대로** 통과시킨다. 각 관문은 앞이 뚫려야 뒤가 의미 있다 (후킹이 없으면 CTA는 존재하지 않는 것과 같다).

1. **3초 후킹** — 첫 장(첫 문장)만 보고 멈추자. 스크롤을 멈출 이유가 있는가? 정보 나열형 제목("~하는 법 5가지")인지, 타깃의 실제 고민 언어인지 판별한다. 후킹이 약하면 나머지 진단과 무관하게 최우선 수정 대상.
2. **타깃 언어 일치** — 만든 사람의 언어(기능·스펙·자랑)로 쓰였는가, 고객이 검색창·DM에 실제로 치는 언어로 쓰였는가. "누구에게"가 특정되지 않은 콘텐츠는 아무에게도 안 닿는다.
3. **흐름 유지** — 중간 장에서 이탈 포인트를 찾는다: 갑자기 어려워지는 장, 글자가 많아지는 장, 앞 장과 연결이 끊기는 장. 각 장의 마지막이 다음 장을 넘기게 만드는지 본다.
4. **전환 설계** — 다 본 사람이 "그래서 뭘 하면 되는데?"에 답이 있는가. CTA가 없거나, 있어도 행동 문턱이 높거나(바로 구매 요구), 콘텐츠 주제와 CTA가 단절되어 있는지 본다. 저장·공유·DM·링크 중 이 콘텐츠에 맞는 전환 1개를 특정한다.
5. **신뢰 요소** — 주장만 있고 증거(후기·수치·비포애프터·근거)가 없는 장을 찾는다. 특히 가격·효과 주장은 증거 없으면 광고 냄새만 남는다.

## 출력 형식

```
## 진단 결과: [콘텐츠 이름]
- 관문별: ① 후킹 △ / ② 타깃언어 ○ / ③ 흐름 ✕ / ④ 전환 △ / ⑤ 신뢰 ○
- 가장 크게 새는 곳: (관문 하나만 지목 + 왜)
- 지금 고칠 1가지: (구체적 수정안 — 대체 문구·장 순서·CTA를 직접 써줄 것)
- 그대로 좋은 것: (바꾸지 말아야 할 강점 1~2개)
```

## 주의
- 5개를 다 고치라고 하지 않는다. **가장 크게 새는 관문 1개**에 수정을 집중시킨다.
- 디자인 취향(색·폰트·감성)은 다루지 않는다 — 반응·전환에 영향을 주는 요소만.
- 수정안은 방향 제시로 끝내지 말고 **대체 문구를 실제로 써서** 준다.
- 진단 대상 콘텐츠에 고객 개인정보·미공개 수치가 있으면 그 부분은 인용하지 않는다.
