## 🎯 미션 1. 내 삶을 돕는 OS 최종 완성
> 지금까지 공유하며 받은 **피드백을 반영해 최종 완성**!
- **완성한 것 (무엇을):**
  **WeatherClothe** — 여행지와 여행 날짜만 입력하면 "그 시기 날씨"와 "그 날씨에 맞는 옷차림"을 한 번에 확인할 수 있는 여행 준비 웹앱. 매번 여행 전에 "OO 몇 월 날씨", "OO 여행 옷차림" 같은 검색을 따로따로 하던 걸 한 화면으로 압축하는 게 목표였다.
  - **날씨 — 감이 아니라 데이터**: Open-Meteo API로 여행지의 최근 10년치 같은 시기 평균 최고·최저기온, 일교차, 강수확률을 계산. 출발이 2주 이내면 실제 예보 데이터도 함께 보여주고, 서울과 기온 차이를 비교해 "몇 도나 더 춥다/덥다"를 바로 체감할 수 있게 함.
  - **목적지 유형 자동 판별**: 도시형 / 자연·액티비티형(그랜드캐년, 융프라우, 파타고니아 등) / 혼합형을 지명 기반으로 자동 분류. 도시 여행과 트레킹 여행은 챙겨야 할 옷이 완전히 다르다는 전제에서 출발한 기능이고, 사용자가 직접 수정도 가능.
  - **일자별 코디 추천**: 스타일(미니멀/캐주얼/클래식/스트릿) × 기온대(한겨울~무더움 6단계) 조합으로 아우터·상의·하의·신발을 매일 다르게 조합해서 추천. 그날 일정을 관광/식사·격식/액티비티 중에서 고르면 코디가 바로 바뀜 — 트레킹 날엔 기능성 아우터·트레킹화로, 격식 있는 자리엔 포인트 아이템이 추가되는 식.
  - **내 사진으로 스타일 분석**: 사진을 업로드하면 캔버스로 색상을 뽑아 뉴트럴/채도 비율을 계산하고 스타일을 자동 추천. AI 비전 API 없이도 "내 옷장 톤"을 반영한 추천이 가능하도록 만든 Phase 1 버전.
  - **여행객 룩 참고 링크 + 도시별 문화 팁**: 지도·호텔·음식 이미지를 걸러낸 구글/핀터레스트 검색 링크와, 도시별(파리·도쿄·방콕 등)·국가별 드레스코드·현지 팁 데이터베이스(성당 방문 시 복장 규정, 우기 대비 등)까지 포함.

- **결과물 (링크·스크린샷 — 이미지는 `이미지첨부/` 폴더에):**
  `WeatherClothe/` 폴더(index.html·app.js·style.css) — 브라우저 하나로 바로 실행되는 완결형 프로토타입. 여행지 입력 → 날씨 탭 / 여행객 룩 탭 / 내 코디 탭 / 팁 탭 4개로 구성된 결과 화면까지 실제로 동작함. TODO: 배포 링크나 스크린샷이 있으면 추가해주세요.
- **알게 된 인사이트:**
  1. **완벽한 재료가 없어도 "그럴듯한 Phase 1"은 만들 수 있다.** AI 비전, 정제된 이미지 DB 없이도 픽셀 색상 값·검색 필터·과거 날씨 데이터 같은 있는 재료를 규칙으로 엮으면 실제로 쓸 수 있는 버전이 나온다.
  2. **한계를 숨기지 않고 명시하는 게 오히려 신뢰를 만든다.** "Phase 1은 색감 기반, Phase 2는 AI 비전"이라고 UI에 직접 적어두니 부족함이 아니라 다음 단계로 가는 로드맵처럼 읽혔다.
  3. **데이터 소스의 언어·형식 불일치는 항상 폴백을 준비해야 한다.** 지오코딩 API가 한글 쿼리를 못 찾거나 언어 설정과 다른 응답을 주는 경우가 실제로 잦았고, 별칭 매핑과 재검색 폴백 없이는 반쪽짜리 기능이 됐을 것이다.
