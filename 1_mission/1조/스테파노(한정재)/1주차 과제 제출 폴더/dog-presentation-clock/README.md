# 🐶 강아지 발표 시계 & 타이머 (크롬 확장)

발표할 때 **현재 시각**(디지털 + 아날로그)과 **남은 시간 카운트다운**을 한 화면에서 보여줍니다.
시계판 배경은 **우리집 강아지 사진**, 시침·분침은 **뼈다귀** 모양이고, 초침은 가는 바늘입니다.
디지털 시계는 시:분만 표시(초 없음)합니다. 카운트다운은 다이얼 테두리의 **진행 링**으로도
표시되며, 시간이 얼마 안 남으면 **노랑 → 빨강**으로 바뀌고 종료 시 **화면 깜빡임 + 알림음**이 울립니다.

## 설치 방법 (빌드 불필요)
1. 크롬 주소창에 `chrome://extensions` 입력 → 이동
2. 오른쪽 위 **개발자 모드(Developer mode)** 켜기
3. **압축 해제된 확장 프로그램을 로드(Load unpacked)** 클릭
4. 이 폴더(`dog-presentation-clock`) 선택
5. 툴바의 🐶 아이콘 클릭 → 시계/타이머 **전용 창**이 열립니다
   (다시 클릭하면 그 창을 앞으로 가져옵니다)

> 코드를 수정했다면 `chrome://extensions`에서 이 확장의 **새로고침(↻)** 버튼을 누른 뒤
> 창을 다시 열면 반영됩니다.

## 사용법
- **분 / 초** 입력칸에 발표 시간을 넣거나, **3분 / 5분 / 10분** 프리셋 버튼 사용
- **시작 / 일시정지 / 리셋** 버튼으로 카운트다운 제어
- **화면(시계 영역) 클릭** = 시작 ↔ 일시정지 토글 (시작/정지 시 효과음)
- **화면 우클릭** = 시계판 사진 바꾸기 팝업 열기
- 🔔 버튼: 알림음 켜기/끄기 (시작·정지·종료음 모두)
- ⛶ 버튼: 전체화면 (발표 화면에 띄우기 좋음)
- 단축키: **Space** = 시작/일시정지, **R** = 리셋

## 우리집 강아지 사진으로 바꾸기 ⭐
**가장 쉬운 방법: 시계 화면에서 마우스 우클릭 → "사진 선택…"** 으로 사진을 고르면
자동으로 정사각형(가운데 기준)으로 잘라 시계판에 적용되고, 브라우저에 저장돼 다음에 열어도 유지됩니다.
("기본 사진으로" 버튼으로 원래 사진 복구)

기본(공유용) 사진을 바꾸려면 파일을 직접 교체해도 됩니다:
1. 강아지 사진 파일 이름을 `dog-bg.jpg`로 바꿔 `assets/` 폴더에 덮어쓰기
2. 확장 새로고침(↻) 후 창 다시 열기
3. `dog-bg.jpg`가 없으면 임시 이미지(`assets/dog.png`)가 대신 표시됩니다.

## 파일 구조
```
dog-presentation-clock/
├── manifest.json      # 확장 설정 (Manifest V3)
├── background.js      # 아이콘 클릭 → 전용 창 열기
├── clock.html         # 화면 구조
├── clock.css          # 디자인 (다크 테마, 경고 색)
├── clock.js           # 시계·타이머 로직 (뼈다귀 침 생성 포함)
├── assets/
│   ├── dog-bg.jpg     # 시계판 배경 사진 (← 여기에 강아지 사진을 넣으세요)
│   └── dog.png        # 배경 사진이 없을 때 쓰는 임시 이미지
└── icons/             # 툴바 아이콘 (16/48/128px)
```

> 알림음은 별도 음원 파일 없이 **Web Audio**로 생성하므로 추가 파일이 필요 없습니다.
