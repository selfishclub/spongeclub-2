# 🐶 강아지 발표 시계 — 초보자용 제작 가이드

크롬 확장을 처음 만드는 사람도 따라 할 수 있도록, 이 프로젝트를 **시작부터 완성까지**
8단계로 정리했습니다.

## 단계별 설명

### 1. 새 폴더 만들기
빈 폴더 하나가 곧 확장 하나입니다. 예: `~/Downloads/dog-presentation-clock`.

### 2. 기본 파일 작성 (빌드 도구 불필요)
| 파일 | 역할 |
|---|---|
| `manifest.json` | 확장 설정 — 반드시 **Manifest V3** |
| `clock.html / clock.css / clock.js` | 시계·타이머 화면과 로직 |
| `background.js` | 아이콘 클릭 시 별도 창 열기 |
| `icons/` (16·48·128px) | 툴바 아이콘 |
| `assets/dog-bg.jpg` | 시계판 배경 강아지 사진 |

초보자가 자주 막히는 점:
- 옛날 자료의 MV2(`browser_action` 등) 말고 **MV3**(`action` + `service_worker`)를 쓴다.
- 인라인 스크립트 금지 — JS는 `.js` 파일로 분리한다.
- `chrome.storage`를 쓰면 `manifest.json`의 `permissions`에 `"storage"`를 넣어야 한다.

### 3. 크롬에 불러오기
`chrome://extensions` → 우상단 **개발자 모드** ON → **압축 해제된 확장 프로그램을 로드**
→ 만든 폴더 선택.

### 4. 기본 시계 확인
툴바의 🐶 아이콘 클릭 → 별도 창에서 디지털 시계·아날로그 시계·카운트다운이
정상 동작하는지 확인.

### 5. 디자인 다듬기 (반복 구간)
강아지 사진을 시계판 배경으로, 바늘을 일반 시계 모양으로, 디지털은 시:분만,
숫자가 있는 바깥 띠는 어둡게(흰 강아지 위 가독성).

### 6. 기능 추가 (반복 구간)
- 화면(시계 영역) 클릭 → 시작/일시정지 토글 + 효과음
- 화면 우클릭 → 사진 교체 팝업(고른 사진을 자동 정사각형 크롭·저장)

### 7. 고치면 새로고침
코드를 수정할 때마다 `chrome://extensions`에서 이 확장의 **새로고침(↻)** 을 누르고
창을 다시 엽니다. **5~6번을 여기서 계속 반복**하며 원하는 모습으로 다듬습니다.

### 8. 발표에서 사용
아이콘 클릭 → **별도 창** → **전체화면(⛶)** 으로 발표 화면에 띄웁니다.
단축키: `Space` = 시작/정지, `R` = 리셋.
