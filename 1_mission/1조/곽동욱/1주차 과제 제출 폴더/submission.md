# 1주차 — 나만의 OS 만들기 🛠️

> 미션을 진행하며 **과정과 결과물**을 기록해주세요. (다 못 채워도 OK, 한 것 위주로!)

## 🎯 미션 1. 내 OS 재료 찾기
> 인터뷰 스킬(아이데이션)로 "내 삶에 필요한 게 뭔지" 찾기
- **과정 (어떻게 찾았나):** 기존에 영어 뉴스레터를 발행하는 전체 과정을 구체적으로 설명하면서 어떤 어려움이 있는지 클로드코드와 찾아갔다. 작게 시작해서 구체적으로 작성하면서 내가 지금 당장 필요한게 무엇인지 정했다.
- **결과:** 이번주에는 '기사 문장 + 표현·단어를 넣으면 원어민 실사용 여부를 바로 알려주는 도구'를 만들어보기로 결정했다.
- **느낀 점:** 클로드코드가 생각보다 똑똑해서 편하고 빠르게 무엇을 만들지 정리해줘서 조금 놀랐다.

## 🧩 미션 2. 내 OS 기획
> 인터뷰 결과 + 세션 내용(흐민·배짱·키노) 활용해 기획
- **기획 내용:** 
1단계 (이번 주) — 텔레그램 표현 검증 봇
기사 읽다가 막히는 표현 발견
→ 텔레그램에 문장 + 단어 던짐
→ 봇이 바로 답: 원어민 실사용 여부 / 캐주얼·비즈니스 구분 / 짝꿍 표현
→ 기사 계속 읽기
흐민처럼 Claude Code 채널 기능으로 텔레그램 봇과 연동하면 돼요.
2단계: 기사 URL 하나 던지면 → 뉴스레터 후보 표현 여러 개를 한꺼번에 뽑아서 검증
3단계: 검증된 표현 + 기사 내용으로 지금 쓰는 제미나이 gem 3단계까지 자동 실행 → 초안 완성
- **막혔던 점 / 어떻게 풀었나:**
고민되었던 부분은 '현재 뉴스기사를 읽다가 제미나이에게 물어보는 과정과 클로드코드가 제안한게 큰 차이가 없는데 하는게 맞을까?'였습니다. 물어보는 걸 제미나이에서 텔레그램으로 옮겨진 것일 뿐 왔다갔다하는 건 마찬가지니까요. 그래도, 클로드와 함께 1단계를 시작하고 개선을 하면서 클로드로 모든 과정을 이관하는 것을 생각하면서 1단계를 먼저 시작하기로 결정했습니다.

## ⚙️ 미션 3. 내 OS 구현
> 실제로 만들어본 것 (클로드코드 '채널' 기능 활용 OK)
- **결과물:** 뉴스기사를 읽던 중 궁금한 단어, 표현을 텔레그램으로 물어보면 답해주는 OS를 만듦. 솔직히 이전에 제미나이에게 물어보던것과 큰 차이는 없음. 하지만, 일상에 조금 우겨넣어볼려고 시도함.
