## 결과물

Chrome 확장 프로그램 **Julie Timer Extension** (v1.0.1) 완성 — `fullscreen-clock-extension/` 폴더.

새 탭을 열면 전체화면 시계가 뜨고, 4가지 모드를 탭으로 전환한다.

- **시계 / Clock** — 현재 시각을 큰 디지털 글씨로, 날짜·요일 표시
- **세계시간 / World** — 12개 도시 중 추가/삭제(서울·뉴욕·런던·도쿄 기본), UTC 오프셋·현지 날짜 표시
- **타이머 / Timer** — 분·초 입력 또는 1·3·5·10·25분 프리셋, 종료 시 알람음(외부 파일 없이 Web Audio), 10초 이하 빨간 경고
- **스톱워치 / Stopwatch** — 1/100초 단위, 랩 기록
- **영/한 전환** — 우측 상단 `EN`/`한` 버튼으로 탭·도시명·날짜·버튼까지 전환, 선택 저장
- **디자인** — 스폰지클럽 노란 그라데이션 + 스폰지 점 텍스처 + 떠오르는 거품, Montserrat 폰트 로컬 번들, 상단 `Philly Julie` 배지
- **단축키** — `1`~`4` 모드 전환, `Space` 타이머/스톱워치 시작·정지

**설치**: `chrome://extensions` → 개발자 모드 ON → 압축해제된 확장 프로그램 로드 → `fullscreen-clock-extension` 폴더 선택 → 새 탭 열기

## 삽질 과정

처음엔 예쁜 단일 HTML 파일 하나를 그대로 확장으로 올리려다 막혔다. 확장은 `manifest.json`이 반드시 있어야 한다는 걸 알게 됐고, HTML·CSS·JS·아이콘을 폴더 구조로 묶고 `manifest.json`에서 새 탭(`chrome_url_overrides.newtab`)을 지정하는 방식으로 정리했다.

폰트(Montserrat)를 Google Fonts CDN으로 불러왔더니, 외부 연결이 계속 열려 있어서 미리보기 캡처가 30초씩 멈췄다. 폰트 woff2 파일을 직접 받아 `fonts/` 폴더에 넣고 `@font-face`로 로컬 참조하도록 바꾸니 해결됐다. 덤으로 **오프라인에서도 돌아가고 로딩 깜빡임도 사라지는** 전화위복이 됐다.

작업 중 확장 폴더가 통째로 사라지는 일도 있었다(OneDrive 동기화로 추정). 파일을 복구한 뒤, 같은 사고를 막으려고 **곧바로 git 커밋·푸시로 백업**해 두는 습관을 들였다.

마지막엔 크롬에 분명 올렸는데 Philly Julie 배지도 안 보이고 버튼도 안 눌렸다. 알고 보니 **크롬에 예전 버전이 로드된 상태**였다. 코드를 고쳐도 자동 갱신이 안 되기 때문에, 확장 카드의 **↻ 새로고침**을 누르고 **새 탭을 다시 열어야** 최신 화면이 나온다는 걸 배웠다. ("may be unsafe" 경고는 웹스토어를 거치지 않은 개발자 모드 확장이라 뜨는 정상 표시였다.)

알람은 mp3를 넣는 대신 Web Audio API 오실레이터로 세 음(660·880·1100Hz)을 울리게 해 파일 없이 처리했고, 타이머는 매초 빼는 대신 **"종료 시각" 기준으로 남은 시간을 계산**해 오차가 안 쌓이게 했다.

## 인사이트

크롬 확장은 인터넷에서 받는 게 아니라 **내 폴더를 크롬에 가리키는 것**이고, 외부 라이브러리 없이 브라우저 내장 API(`Intl`·`Web Audio`·`requestAnimationFrame`·`localStorage`)와 로컬에 번들한 에셋만으로도 충분히 완성도 있는 도구를 만들 수 있다. 그리고 **만든 즉시 git에 올려 백업하는 습관**이 결과물보다 더 큰 자산이라는 걸 몸으로 배웠다.
