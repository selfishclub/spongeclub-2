## 결과물

Chrome 확장 프로그램 **Fullscreen Clock & Timer** 완성 (`fullscreen-clock-extension/` 폴더)

- **시계** — 현재 시각을 큰 디지털 글씨로, 날짜·요일 표시
- **세계시간** — 12개 도시 중 원하는 도시 추가/삭제, 설정 자동 저장
- **타이머** — 분/초 직접 입력 또는 1·3·5·10·25분 프리셋, 종료 시 알람음
- **스톱워치** — 1/100초 단위, 랩 기록
- 한/영 전환, 단축키(`1`~`4` 모드 전환, `Space` 시작·정지)

설치: `chrome://extensions` → 개발자 모드 ON → 압축해제된 확장 프로그램 로드 → 폴더 선택

## 삽질 과정

타이머 알람을 어떻게 구현할지가 제일 막혔다. 처음엔 `<audio>` 태그에 mp3 파일을 넣으려 했는데, 외부 파일을 번들에 포함하면 용량도 늘고 manifest 권한도 복잡해질 것 같아서 다른 방법을 찾았다. Web Audio API로 오실레이터를 직접 만들면 파일 없이 소리를 낼 수 있다는 걸 알고 적용했더니 깔끔하게 해결됐다. 세 가지 주파수(660·880·1100Hz)를 0.18초 간격으로 울리게 하니 딱 맞는 느낌이 났다.

세계시간에서 각 도시의 UTC 오프셋 표시도 생각보다 까다로웠다. `Intl.DateTimeFormat`의 `timeZoneName: "shortOffset"` 옵션으로 "GMT+9" 같은 문자열을 뽑아낸 뒤 "GMT"를 "UTC"로 바꿔서 보여주는 방식으로 해결했다.

## 인사이트

외부 라이브러리 없이 브라우저 내장 API(`Intl`, `Web Audio`, `requestAnimationFrame`, `localStorage`)만으로도 생각보다 훨씬 풍성한 기능을 만들 수 있다.
