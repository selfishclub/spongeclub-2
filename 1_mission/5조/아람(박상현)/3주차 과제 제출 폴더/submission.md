## 🎯 미션 1. 내 삶을 돕는 OS 최종 완성
> 지금까지 공유하며 받은 **피드백을 반영해 최종 완성**!

- **완성한 것 (무엇을):**
  - **보안 3계층 뼈대** — 데이터를 🟢공개 / 🟡로그인 / 🔴비밀 로 나눴다. 특히 **재무·매출(🔴)은 배포에서 아예 제외**하고 내 PC(로컬)에서만 보이게 하는 파이프라인(`tiers.js` + `_secret/` 폴더 + `/vault` 금고)을 만들었다.
  - **하드코딩 비번 제거** — 관리자(`/admin`) 암호가 코드에 `arami`로 박혀 있던 걸 지우고 `.env` 기반으로 교체.
  - **자동 셸 선물 봇을 OS에 편입** — 매일 사라지는 셸을 조원에게 공정하게 돌아가며 선물하는 봇을, 지도·대시보드·**셸 스튜디오(`/shell-studio`)**로 등록. 봇은 **웹(클라우드)·PC 앱** 두 방식으로 만들고, 남들도 쓸 수 있게 매뉴얼까지 붙였다.
  - **대시보드(관제실) 방 진입 404 수정** — 정적 페이지 링크 문제로 '대시보드'·'방' 진입 시 Not Found 나던 것 해결.

- **피드백 반영한 점:**
  - *"회사 자료(재무·매출)까지 넣되 절대 새면 안 된다"* → 정적 사이트에선 클라이언트 비밀번호가 보안이 아님(유리문에 친 커튼)을 확인하고, **비번으로 가리는 대신 데이터를 빌드에서 물리적으로 제외**하는 구조로 바꿨다.
  - *"됐다는 실제로 눌러봐야 진짜 된 거다"(2주차 교훈)* → 이번엔 배포본을 grep하고 모든 경로의 **응답코드를 직접 찍어** 검수했다.

- **결과물 (링크·스크린샷 — 이미지는 `이미지첨부/` 폴더에):**
  - 아라미러 v2 (로컬 dev에서 완성, 배포 반영은 대기): 주요 페이지 `/vault`(🔴 금고), `/shell-studio`(셸 스튜디오), `/hq`(관제실)
  - 검증: `npm run build` 95페이지 초록 · 배포 산출물(dist) 전체 grep → **재무 문자열 0건**, 옛 비번 `arami` **0건**
  - 봇 실제 코드: `Project/자동 셸 선물 봇` (config·명단만 바꾸면 1~6조 누구나 사용)

- **알게 된 인사이트:**
  - **보안은 예쁜 자물쇠 UI가 아니라, 데이터가 물리적으로 어디 사는가다.** 비번을 몇 겹 걸었느냐가 아니라, 애초에 그 값이 배포본에 있느냐 없느냐가 진짜다.
  - **"철저히 막아줘"에 대충 막는 척 대신 "그 방식으론 안 됩니다"라고 정직하게 말해주는 게 진짜 도움이다.** 편한 거짓말보다 불편한 사실.
  - **검수는 "빌드 초록"이 아니라 "실제로 클릭(=증거)"이다.**

## 📣 미션 2. 스폰지 토크데이 SNS 후기
> 오늘 토크데이 후기를 SNS에 올리기 (**#스폰지클럽 필수 · 셀 3개 지급!**)

- **후기 내용:**

  > 이번 주 저는 제 개인 OS(아라미러)에 **자동 셸 선물 봇**을 붙였습니다. 스폰지클럽에선 매일 셸이 하나씩 생겼다가 하루 지나면 사라지는데, 자꾸 잊거나 한 사람에게만 몰리더라고요. 그래서 나를 뺀 조원에게 **매일 공정하게 돌아가며** 셸을 선물하도록 자동화했어요.
  >
  > 재밌는 건 이걸 **두 가지 방식**으로 만들었다는 거예요. 하나는 **웹(클라우드)** — 제 PC가 꺼져 있어도 매일 "오늘은 누구 차례"라고 알려주고, 다른 하나는 **제 PC에서 도는 앱** — 전송까지 완전 자동입니다. 둘 다 한곳에 담아, 남들도 조·명단만 바꾸면 쓰도록 매뉴얼까지 정리했어요.
  >
  > 그리고 회사 자료를 OS에 넣으려다 *"정적 웹사이트에 비밀번호 걸면 안전한 게 아니다"*라는 것도 배웠습니다. 편한 거짓말보다 불편한 사실을 말해주는 게 진짜 도움이라는 것도요.
  >
  > 혼자였으면 여기까지 못 왔을 텐데, 크루들과 같이 부딪히니 훨씬 멀리 갑니다.
