# 🐚 자동 셀 선물 봇

스폰지클럽에서 **매일 하나씩 생기는 셀**은 하루가 지나면 사라집니다.
잊지 않고, 그리고 **한 사람에게 몰리지 않게** 조원들에게 공정하게 돌아가며 셀을 선물하도록 도와주는 봇이에요.

> 5조 기준으로 만들었고, 1~6조 누구나 설정만 바꿔 쓸 수 있습니다.

---

## 🧠 어떻게 돌아가나요?

- 나(owner)를 뺀 조원들을 **매일 한 명씩 순서대로** 돌아가며 대상으로 정합니다.
- 5조는 11명이라 **11일이면 모두 한 번씩**, 그다음 다시 처음부터 반복돼요.
- 좋아하는 사람 5~6명만 정해두면 (`favorites` 모드) **그 안에서만** 순환할 수도 있어요.
- 매일 준 기록(`state.json`)이 남아서, 껐다 켜도 순서가 이어집니다.

**실제 명령어:** `/셸보내기 @닉네임 코멘트` (스폰지셸봇)
`셀`이 아니라 **`셸`**, 띄어쓰기 없이 `/셸보내기` 입니다. 참고로 `/잔고` 로 내 셸 잔고를 볼 수 있어요.

**핵심 제약(꼭 알아두세요):**
`/셸보내기` 는 Slack의 *슬래시 명령* 이라, **봇/API가 대신 실행할 수 없습니다.**
그래서 '마지막 전송'을 두 가지 방식 중에 고를 수 있게 만들었어요.

| | 방식 A · 리마인더 | 방식 B · 브라우저 완전자동 |
|---|---|---|
| 자동화 정도 | 대상 계산 + 알림까지 자동, **전송 1탭은 내가** | **0탭 완전 자동** |
| 안정성 | 매우 높음 | Slack 화면 바뀌면 보정 필요 |
| 실행 위치 | GitHub 클라우드(내 PC 꺼져 있어도 됨) | **내 PC가 그 시간에 켜져 있어야 함** |
| 남들과 공유 | 쉬움 (레포 복제해서 설정만) | 각자 자기 PC에 설치해야 함 |

두 방식이 **한 레포에 다 들어있습니다.** 원하는 걸 쓰면 돼요.

---

## ⚙️ 설정 (`config.json`)

```json
{
  "team": "5조",
  "owner_nickname": "아람",          // 나 (선물 보내는 사람) — 이 사람은 대상에서 제외
  "mode": "rotate_all",              // rotate_all = 전체 순환 / favorites = 선호 인원만
  "favorites": ["그레이", "포비", "리카", "필리줄리", "채리"],
  "comments": ["오늘도 화이팅입니다! 🐚", "..."]  // 매일 코멘트 자동 선택
}
```

- 다른 조로 바꾸려면 `team` 값과 `data/members.json` 만 채우면 됩니다.
- **오늘 누가 대상인지 미리보기:** `python rotation.py`

---

## 🅰️ 방식 A · 리마인더 (추천, 클라우드 자동)

매일 00:01(KST)에 봇이 **나에게 Slack DM**으로 이렇게 보내줍니다:

> :shell: 오늘은 **그레이(김수현)** 님 차례예요!
> ```/셸보내기 @그레이 오늘도 화이팅! 🐚```

나는 5조 채널에서 한 번만 보내면 끝. (`@그레이`는 입력창에서 멘션으로 골라주세요.)

**설치 (한 번만):**
1. Slack 앱에서 봇 토큰 발급 → 권한 `chat:write`, `im:write`
2. 이 레포 → **Settings → Secrets and variables → Actions → New secret**
   - 이름 `SLACK_BOT_TOKEN`, 값에 발급받은 토큰(`xoxb-...`)
3. 끝! 매일 자동 실행됩니다. (**Actions 탭 → 매일 셀 리마인더 → Run workflow** 로 즉시 테스트 가능)

로컬에서 미리보기: `python reminder.py --dry-run`

---

## 🅱️ 방식 B · 브라우저 완전자동 (내 PC 전용, 0탭)

내 PC에서 매일 밤 Slack을 실제로 열어 명령을 입력·전송합니다.

**설치 (한 번만):**
```bash
pip install playwright
playwright install chromium
python auto_send.py --login      # 창에서 Slack 로그인 → 콘솔 Enter (세션 저장됨)
```

**매일 자동 실행 (Windows 작업 스케줄러):**
1. `작업 스케줄러` 열기 → **기본 작업 만들기**
2. 트리거: **매일**, 시간 **오전 12:01**
3. 동작: **프로그램 시작** → `run_auto.bat` 선택 (이 폴더 안에 있음)
4. 확인. 이제 PC가 켜져 있으면 매일 밤 자동 전송됩니다.

먼저 눈으로 확인: `python auto_send.py --headful`

## 🗂 파일 구조

```
config.json            내 설정 (조, 모드, 코멘트)
data/members.json      조별 명단 + Slack ID
rotation.py            공정 순환 엔진 (누구 차례인지 계산)
reminder.py            방식 A: Slack DM 리마인더
auto_send.py           방식 B: 브라우저 완전자동
run_auto.bat           방식 B를 작업 스케줄러로 돌릴 때 쓰는 실행 파일
state.json             순환 기록 (자동 생성)
.github/workflows/     방식 A 매일 자동 실행 설정
```

## ❓ 자주 묻는 것

- **한 사람한테 몰리지 않나요?** — 안 몰립니다. 가장 오래 안 준 사람부터 골라 완전히 골고루 돕니다.
- **하루 걸렀어요.** — 괜찮아요. 순서는 기록 기준이라 다음 사람으로 자연스럽게 이어집니다.
- **다른 조도 쓰고 싶어요.** — `config.json`의 `team`과 `data/members.json`만 채우면 됩니다.
