# 1주차 — 나만의 OS 만들기 🛠️

## 🎯 미션 1. 내 OS 재료 찾기
> 인터뷰 스킬(아이데이션)로 "내 삶에 필요한 게 뭔지" 찾기
- **과정 (어떻게 찾았나):** 하루 중 시간을 제일 많이 뺏긴 일을 떠올리는 것부터 시작 — 컴퓨터가 자꾸 말썽이라 결국 새 랩탑까지 산 이야기가 먼저 나왔다. 거기서 한 발 더 들어가서 "할 때마다 짜증 나는 반복 업무"를 짚어보니, 고객이 매물 쇼잉을 요청할 때마다 에이전트한테 전화해서 커미션 스플릿을 확인하고 리포트로 정리해 보내주는 과정이 나왔다. 일 얘기만 하다가 삶 쪽으로도 한 번 짚어봤는데(다이어트), 비교해보니 지금 하루를 더 가볍게 만드는 건 업무 쪽이라는 결론.
- **결과:** 진짜 문제는 리포트 자체가 아니라 "전화 거는 것" 그 자체였다. 부동산 에이전트뿐 아니라 신용카드 회사, PECO(전기회사)까지 — 전화만 걸면 다 싫다는 게 핵심 마찰점으로 드러났다. 요즘은 상대편도 AI가 응답해서 라이브 통화 자체가 더 힘들어졌다는 점도 확인.
- **느낀 점:** "딱히 불편한 거 없는데"로 시작했는데, 하루를 한 겹씩 벗겨보니 전화 자체에 대한 스트레스가 계속 반복되고 있었다는 걸 알았다. 문제를 좁혀서 다시 물어보니(리포트 → 전화) 훨씬 구체적인 재료가 나왔다.

```
🟡 내 삶을 돕는 OS · 재료 카드

· 영역      : 일 (부동산 리로케이션 컨설팅 — 고객 쇼잉 리포트 업무)
· 걸리는 지점 : 리포트를 만들려면 매번 전화를 걸어야 하는데, 그 전화 거는 것 자체가 너무 싫음
              (에이전트뿐 아니라 신용카드사, PECO까지 — 요즘은 AI가 응답해서 라이브 통화도 어려움)
· 지금은    : 고객이 Zillow에서 매물 가져옴 → 에이전트한테 전화해서 커미션 스플릿 확인
              (여기서 시간 제일 많이 걸림) → 고객 상황 컨펌 → 이메일로 확인받아 정리
              → 고객에게 리포트 발송 → 피드백 오면 처음부터 반복
· OS가 된다면 : AI가 대신 전화를 걸어서 필요한 정보(커미션 스플릿 등)를 받아와줌
· 한 문장   : "싫어하는 일을 안 해도 되는 하루가 된다"
· 첫 한 걸음 : AI 보이스 에이전트(전화 거는 AI) 툴 리서치 + 통화 스크립트 자동 준비부터 시작
```

## 🧩 미션 2. 내 OS 기획
> 인터뷰 결과 + 세션 내용 활용해 기획
- **기획 내용:**
  - 무엇을: 고객 쇼잉 리포트를 만들 때마다 거쳐야 하는 "전화 거는 일"을 AI 음성 에이전트가 대신하는 **전화 대행 OS**
  - 왜: 문제는 리포트 정리가 아니라 전화 자체 — 전화가 싫어서 미루거나 시간을 많이 뺏김
  - 3단계로 쪼갬:
    - **1단계 (이번 주)**: 본인 인증이 필요 없는 통화부터 — AI가 에이전트에게 전화해서 매물 커미션 스플릿을 확인해오는 것. 매물 주소를 넣으면 AI가 전화를 걸고 결과를 알려줌.
    - **2단계**: 통화 결과를 자동으로 고객용 리포트 이메일 초안으로 정리
    - **3단계**: 본인 인증이 필요한 통화(PECO, 신용카드사 등)까지 확장 — 계정 정보를 AI에게 안전하게 넘기는 방법 필요
  - 설계 원칙: 민감한 본인 인증이 필요 없는 통화부터 안전하게 시작하고, 계정 인증이 걸리는 통화는 보안 검토 후 나중 단계로
- **막혔던 점 / 어떻게 풀었나:** 처음엔 "PECO도 대신 걸어줘" 하려 했는데, 생각해보니 본인 인증(계좌번호·생년월일 등)이 걸린 통화는 AI가 인증을 통과 못 하거나 보안상 위험할 수 있다는 걸 깨달았다. 그래서 인증이 필요 없는 "정보성 통화"(에이전트 커미션 확인)부터 먼저 만들고, 인증이 필요한 통화는 3단계로 미루는 것으로 범위를 좁혔다.

## ⚙️ 미션 3. 내 OS 구현
> 실제로 만들어본 것

**① 새 탭 시계 확장 프로그램 (완성)**

Chrome 확장 프로그램 **Fullscreen Clock & Timer** (`fullscreen-clock-extension/` 폴더)

- **시계** — 현재 시각을 큰 디지털 글씨로, 날짜·요일 표시
- **세계시간** — 12개 도시 중 원하는 도시 추가/삭제, 설정 자동 저장
- **타이머** — 분/초 직접 입력 또는 1·3·5·10·25분 프리셋, 종료 시 알람음
- **스톱워치** — 1/100초 단위, 랩 기록
- 한/영 전환, 단축키(`1`~`4` 모드 전환, `Space` 시작·정지)

설치: `chrome://extensions` → 개발자 모드 ON → 압축해제된 확장 프로그램 로드 → 폴더 선택

**② AI 전화 에이전트 — 커미션 스플릿 확인 콜 (1단계 착수, Vapi 계정 연동 대기 중)**

미션 1·2에서 나온 "전화 대행 OS"의 1단계 코드 (`ai-call-agent/` 폴더)

- 매물 주소를 넣으면 [Vapi](https://vapi.ai) AI 음성 에이전트가 부동산 에이전트에게 전화를 걸어 커미션 스플릿·특이사항을 확인
- `assistant-config.json` — AI 통화 스크립트(시스템 프롬프트, 첫 인사말)
- `make-call.js` — Vapi API로 실제 통화를 트리거하는 스크립트
- 아직 Vapi 계정 가입·전화번호 구매가 남아있어서, 계정 준비되는 대로 바로 실제 통화 테스트 가능한 상태로 코드는 완성해둠 (자세한 설정법은 `ai-call-agent/README.md`)

**결과물 / 링크:**
- `fullscreen-clock-extension/` — 완성, 바로 설치 가능
- `ai-call-agent/` — 코드 완성, 계정 연동 대기

## 삽질 과정
> 1주차 전체를 돌아보며

시계 확장에서는 타이머 알람을 어떻게 구현할지가 제일 막혔다. 처음엔 `<audio>` 태그에 mp3 파일을 넣으려 했는데, 외부 파일을 번들에 포함하면 용량도 늘고 manifest 권한도 복잡해질 것 같아서 다른 방법을 찾았다. Web Audio API로 오실레이터를 직접 만들면 파일 없이 소리를 낼 수 있다는 걸 알고 적용했더니 깔끔하게 해결됐다. 세 가지 주파수(660·880·1100Hz)를 0.18초 간격으로 울리게 하니 딱 맞는 느낌이 났다.

세계시간에서 각 도시의 UTC 오프셋 표시도 생각보다 까다로웠다. `Intl.DateTimeFormat`의 `timeZoneName: "shortOffset"` 옵션으로 "GMT+9" 같은 문자열을 뽑아낸 뒤 "GMT"를 "UTC"로 바꿔서 보여주는 방식으로 해결했다.

AI 전화 에이전트 쪽에서는 "AI가 어떻게 본인 인증을 통과하지?"라는 질문에 부딪혔다. PECO나 카드사처럼 계정 인증이 필요한 곳까지 욕심내다가, 인증 없는 통화부터 안전하게 시작하는 쪽으로 범위를 좁힌 게 이번 주의 중요한 판단이었다.

## 인사이트

외부 라이브러리 없이 브라우저 내장 API(`Intl`, `Web Audio`, `requestAnimationFrame`, `localStorage`)만으로도 생각보다 훨씬 풍성한 기능을 만들 수 있다. 그리고 "딱히 불편한 거 없는데"로 시작해도, 하루를 한 겹씩 벗겨보면 반복되는 마찰점(전화 거는 것)이 반드시 나온다 — 문제를 좁혀서 다시 물어보는 게 핵심이었다.
