# 1주차 — 나만의 OS 만들기 🛠️

> 미션을 진행하며 **과정과 결과물**을 기록해주세요. (다 못 채워도 OK, 한 것 위주로!)

## 🎯 미션 1. 내 OS 재료 찾기
> 인터뷰 스킬(아이데이션)로 "내 삶에 필요한 게 뭔지" 찾기
- **과정 (어떻게 찾았나):** 인터뷰로 하루를 훑어보니, **매달 반복하는 업무 데이터 점검(클리닝)**이 시간을 제일 많이 잡아먹는 일이었습니다. 그게 확실한 재료였어요.
- **결과:** "매월 반복하던 데이터 점검을 자동으로 돌려주는, 내 일을 돕는 OS"를 만들기로 정했습니다.
- **느낀 점:** "딱히 만들 게 없다"고 생각했는데, 막상 들여다보니 **반복되는 사소한 수작업이 가장 좋은 재료**였습니다.

## 🧩 미션 2. 내 OS 기획
> 인터뷰 결과 + 세션 내용(흐민·배짱·키노) 활용해 기획
- **기획 내용:** 엑셀을 넣으면 → 정해둔 **규칙으로 자동 점검** → ① 건별 점검 결과(중요도 2단계로 구분, 항목 클릭 시 상세) + ② 담당자에게 공유할 **통보용 보고문**을 자동으로 만들어 주는 앱. **브라우저 안에서만** 동작해 데이터가 외부로 안 나가고, 오프라인에서도 돌게 설계했습니다.
- **막혔던 점 / 어떻게 풀었나:** ① 제가 고치는 곳은 데이터 파일이 아니라 **별도 시스템**이라, 결과물이 "고친 데이터"가 아니라 **"무엇을 고쳐야 하는지 알리는 보고문"**이어야 한다는 걸 깨닫고 방향을 바로잡았습니다. ② **개인정보 보안**이 가장 큰 고민이었는데, 데이터를 클라우드·AI로 보내지 않고 **브라우저 로컬에서만 처리**하는 방식으로 풀었습니다.

## ⚙️ 미션 3. 내 OS 구현
> 실제로 만들어본 것 (클로드코드 '채널' 기능 활용 OK)
- **결과물:** 위 기획대로 **검증 앱을 직접 완성**했습니다. 엑셀을 넣으면 여러 점검 규칙으로 자동 검사하고, 케이스별 결과와 통보용 보고문을 만들어 줍니다. 손으로 하던 보조 작업까지 앱이 자동으로 대신하도록 했습니다.
- **링크 / 스크린샷:** ⚠️ 다루는 데이터가 **개인정보가 포함된 사내 업무 데이터**라, **데이터 구조·검증 규칙이 외부에 드러나면 안 됩니다.** 그래서 코드·구조는 공개하지 않고, **시연 또는 민감 정보를 가린 부분 스크린샷으로 갈음 인증**을 요청드립니다. (개발은 가짜 샘플로만 진행, 실제 데이터는 본인 PC에서만 처리)
