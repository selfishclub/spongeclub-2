# 1주차 — 나만의 OS 만들기 🛠️

> 미션을 진행하며 **과정과 결과물**을 기록해주세요. (다 못 채워도 OK, 한 것 위주로!)

## 🎯 미션 1. 내 OS 재료 찾기
> 인터뷰 스킬로 "내 삶에 필요한 것" 찾기
- **과정 (어떻게 찾았나):** 클로드와 대화하며 1주차 과제에서 자꾸 미루는 일이 무엇인지 물어봤다. 자료 찾기와 정리 기준이 헷갈릴 때 시작이 느려진다는 점을 확인했다.
- **결과:**
  - 영역: 삶
  - 걸리는 지점: 1주차 과제 자료를 어디에 어떻게 정리해야 할지 헷갈려서 자꾸 미루는 것
  - 지금은: 필요한 내용을 포스트잇에 적어 보이는 곳에 붙여둔다. 시간이 지나면 이전 것은 잘 보지 못하고, 내용이 흩어지거나 사라진다.
  - OS가 된다면: 분류 기준을 제시해주고, 정보들을 한곳에 모아 쉽게 다시 찾아보고 아이디어로 이어지게 해주는 도구
  - 한 문장: "중요한 생각과 자료를 잃어버리지 않고, 다시 꺼낼 수 있게 정리된다"
- **느낀 점:** 정리 기준과 공간이 불분명하면 시작부터 막힌다. 작은 정리 공간과 단순 분류 틀부터 먼저 만드는 것이 더 도움이 된다.

## 🧩 미션 2. 내 OS 기획
> 인터뷰 결과와 세션 내용을 활용해 기획
- **기획 내용:** 1주차 과제 전용 정리 공간을 만든다. 가장 단순한 분류 틀부터 적용해 ‘어디에 넣을지’ 고민을 줄인다.
  - 예시 템플릿: "주제 / 출처 / 다음 행동"
  - 또는: "아이디어 / 참고 자료 / 다시 볼 때"
- **막혔던 점 / 어떻게 풀었나:** 포스트잇은 유지가 어렵고 정리 기준이 모호했다. 그래서 먼저 과제 전용 정리 방식과 최소한의 분류 틀을 정해, 다시 시작할 수 있는 기반을 만들었다.

- **구현 아이디어:** 과제 아이디어와 참고 자료를 한 곳에 모으고, 나중에 다시 볼 때 기준을 정해두는 방식으로 정리한다.
