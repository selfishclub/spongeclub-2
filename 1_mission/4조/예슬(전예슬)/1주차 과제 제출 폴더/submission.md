# 1주차 — 나만의 OS 만들기 🛠️

> 미션을 진행하며 **과정과 결과물**을 기록해주세요. (다 못 채워도 OK, 한 것 위주로!)

## 🎯 미션 1. 내 OS 재료 찾기
> 인터뷰 스킬(아이데이션)로 "내 삶에 필요한 게 뭔지" 찾기
- **과정 (어떻게 찾았나):** 인터뷰 스킬을 클로드코드에서 발동해, "내 일에서 뭐가 제일 막막한가"를 단계적으로 파고들었다. 스킬이 영역 → 걸리는 지점 → 지금 하는 방식 → OS가 되면 어떤 모습인지 순으로 질문을 던져줘서, 막연한 불편을 구체적인 재료 카드로 좁혔다.
- **결과:** "AI 트렌드 리서치"가 내 OS 재료로 나왔다.
  - 영역: 일 (AI 트렌드 리서치)
  - 걸리는 지점: 트렌드 조사가 "어디 볼지 정하고 → 흩어진 걸 모으고 → 정리·요약"까지 세 단계 다 매번 막막함.
  - 지금은: 사이트·SNS를 그때그때 들어가 눈에 띄는 걸 긁어모아 문서에 손으로 정리.
  - OS가 된다면: 빈도·순위가 매겨진 공개 소스(Google Trends·Reddit·YouTube)에서 자동 수집 → "이번 주 트렌드 3줄 요약"까지 한 번에.
- **느낀 점:** 막막함의 정체가 "정보가 없어서"가 아니라 "매번 어디서·어떻게 모을지 처음부터 정해야 해서"라는 걸 인터뷰로 깨달았다. 자동화할 대상이 "조사"가 아니라 "조사의 반복 절차"라는 게 분명해졌다.

## 🧩 미션 2. 내 OS 기획
> 인터뷰 결과 + 세션 내용(흐민·배짱·키노) 활용해 기획
- **기획 내용:** 트렌드 리서치 OS를 "출처 → 수집 → 요약" 3단계로 잡고, 이번엔 수집+요약을 한 칸으로 먼저 만든다. 출처는 인스타·X(로그인·API 막힘) 대신 빈도가 이미 매겨진 공개 소스(Google Trends=검색량, Reddit=upvote, YouTube=조회수)로 정했다. 이 세 곳에서 자동 수집해 "이번 주 트렌드 3줄"로 요약하는 파이프라인 하나가 1단계 목표.
- **막혔던 점 / 어떻게 풀었나:** ①설계 단계: 처음엔 인스타·X 같은 SNS를 긁어 트렌드를 잡으려 했으나 스크래핑이 막혔다. → "빈도가 이미 매겨진 공개 소스"로 방향을 틀었다.
  ②구현 단계: 스킬 마켓 Grade A 추천 스킬(content-trend-researcher)을 깔았는데 데이터가 전부 시뮬레이션(가짜)이라 삭제하고 직접 만들었다. Reddit은 WebSearch·API로 막혀 RSS로 우회(연속요청 429는 sleep 9~15초로 해결), Google Trends는 핵심 기능이 429 차단이라 제외했다. 틱톡·메타 광고라이브러리도 토큰·봇챌린지로 막혀, 뚫리는 소스(YouTube 한국어 조회수·검색 자동완성)로 대중 축을 채웠다.
