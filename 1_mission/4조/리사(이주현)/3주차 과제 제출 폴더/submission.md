## 🎯 미션 1. 내 삶을 돕는 OS 최종 완성
> 지금까지 공유하며 받은 **피드백을 반영해 최종 완성**!

**✅ 완성한 OS:** **로빌(ROBIL)** — *Body를 Log하고, 매일 Buildup 한다.*
사진 한 장·말 한마디로 가볍게 기록하고, 순수 매크로(단백질·탄수·지방 g)로 정직하게 분석하고, **내 몸의 진짜 맥락(호르몬·생리주기)에 맞춰** 영양·운동·휴식을 코칭해주는 **개인용 AI 헬스 코치**. 서버 없이 `index.html` 파일 하나로 돌아가고, 내 기록은 전부 내 브라우저 안에만 저장된다. 이번 주에 실제 인터넷 주소로 **배포**해 아이폰 홈 화면에 앱처럼 올렸다.

- **완성한 것 (무엇을):**
  - **오늘 탭** — 아침 루틴(수면 스샷 판독·공복 체중·컨디션·아침 분석) → 일정 → 운동 → 끼니(사진/텍스트 → 순수 매크로 자동 계산). "지금까지 먹은 것 + 자기 전까지 더 먹을 수 있는 양"을 함께 보여줌
  - **운동 탭** — 구글 캘린더(모든 캘린더)에서 이번 주 운동을 끌어와 밸런스 체크. 탄수 높았던 밤엔 '자기 전 글리코겐 소모 운동'과 **실제** 유튜브 영상(공식 API로 검증한 것만) 추천
  - **그래프 탭** — 공복 체중·체지방률 추이 + 목표까지 요약
  - **인바디 탭** — 결과지 **사진을 찍으면 수치를 자동 판독**해 날짜별 기록 + 건강 진화 그래프
  - **설정 탭** — 내 정보·호르몬 컨텍스트·음식/영양제 사전·식사 루틴·구글 캘린더·AI 키·백업
  - **AI 코치 챗** — 오늘 데이터를 맥락으로 읽고, 대화만으로 기록을 추가·수정·삭제까지 (영양제 성분 코칭은 '의사 상의' 전제)
  - **기술:** 단일 `index.html` + Netlify(HTTPS) / 데이터 주권(기록은 localStorage, 사진은 IndexedDB, 전부 내 기기 안) / 브라우저에서 Claude API 직접 호출(비전·`json_schema`·tool use) / 키는 브라우저에만 저장, 배포 파일엔 미포함 → **월 운영비 $0**

- **피드백 반영한 점:** (실제로 써보며 드러난 불편·피드백을 하나씩 반영해 '진짜 쓰는 앱'으로 다듬음)
  - **"입력이 번거로우면 결국 안 쓴다"** → 사진·텍스트·탭 선택 다 허용하고, "안 먹었어요"와 몰아쓰기까지 기록으로 인정
  - **"숫자만 보여주면 소용없다"** → 메뉴를 강요하는 대신 *"닭가슴살 130g = 소고기 안심 150g = 고등어 한 마리"* 처럼 **바꿔 먹을 g**로 제안
  - **아이폰에서 채팅 입력창이 확대돼 '보내기'가 화면 밖으로 밀림** → 입력창 글자를 16px로 고정해 해결 / "저장됨" 알림이 키보드에 가려지던 것도 입력칸 위 인라인 표시로 개선
  - **캘린더 운동이 일부 안 딸려옴** → `primary`만 보던 걸 **모든 캘린더 병합 조회**로 수정
  - **AI가 추천한 유튜브 링크가 죽은 링크투성이(환각)** → 유튜브 공식 Data API로 **실제 영상만** 노출
  - **"무리하게 만들면 지친다"** → 죄책감 없는 **'여유일'** 과, 이름을 부르며 담백하게 말하는 코치 톤으로 정리

- **결과물 (링크·스크린샷 — 이미지는 `이미지첨부/` 폴더에):**
  - **🔗 라이브 앱:** https://euphonious-taiyaki-a6bc5c.netlify.app/ (아이폰 홈 화면에 추가해 앱처럼 사용 / 키는 각자 브라우저에만 저장돼 링크를 열어도 남의 건강 데이터는 안 보임)
  - **🖼 발표 덱:** https://claude.ai/code/artifact/5d8f0818-90c8-47ac-8fb5-fdb776a5ff04 (로빌 소개·컨셉·기능·아키텍처·삽질·로드맵 10장)
  - **스크린샷:** _(앱 화면 캡처를 `이미지첨부/` 폴더에 추가 예정)_

- **알게 된 인사이트:**
  - **숫자를 모으는 앱이 아니라 '방향을 쌓는 앱'을, 비개발자인 내가 직접 만들 수 있었다.** 시중 앱이 절대 안 넣어주는 '내 몸의 진짜 맥락(호르몬·주기)'과 '그래서 뭘 먹지/하지'까지, **데이터 주권(전부 내 기기 안)을 지키면서** AI 코칭으로 풀 수 있다는 걸 확인했다.
  - **제일 큰 배움은 기술이 아니라 태도였다.** 구글 드라이브가 원본 폴더를 삼켜 실수로 삭제까지 갔다가 휴지통에서 깃 기록째 복구한 뒤로, '자주 커밋하고, 배포본은 따로 둔다'가 몸에 뱄다. ("코드는 멀쩡한데 왜 안 되지"의 답이 코드 바깥 설정에 있을 때가 많았다 — 서비스계정 강제, OAuth 원본 등록, json_schema strict 같은 것들.)
  - **만든 데 쓴 스킬·도구:** `superpowers:brainstorming`(컨셉), `frontend-design`(UI), `claude-api`(브라우저 직접 호출), `artifact-design`(발표 덱) + Claude Design(DesignSync MCP)로 팔레트·목업(웜 뉴트럴 + 발레 로즈). 새 커스텀 스킬은 만들지 않고 **기존 스킬 조합으로 완성**.
