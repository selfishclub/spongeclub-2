## 🎯 핵심 기능 (MVP)

### **1단계: 모임 관리**
- ✅ 모임 생성 (모임명, 설명, 설정)
- ✅ 하이브리드 멤버 초대
  - **멤버:** 영구 등록, 항상 표시, 초대 링크 공유
  - **게스트:** 일회성, 그 라운드에만 표시, QR/링크로 진입
- ✅ 멤버 최대 20명 제한
- ✅ 모임별 랭킹 기준 설정 (스코어 vs 핸디 선택)

### **2단계: 라운드 기록**
- ✅ 라운드 생성 (날짜, 코스명, Par 정보)
- ✅ 스코어카드 입력
  - **방법 A:** 홀별 수동 입력 (1-18홀 각각)
  - **방법 B:** 스코어카드 사진 인식 (OCR) → 자동 입력
- ✅ 각 멤버/게스트별 18홀 스코어 기록

### **3단계: 핸디캡 자동 계산**
- ✅ 최근 3라운딩 스코어 평균
- ✅ 핸디 = (평균 스코어 - Par)
- ✅ 실시간 업데이트

### **4단계: 랭킹 시스템**
- ✅ **스코어 기준:** 총 스코어 낮은 순
- ✅ **핸디 기준:** 스코어 - 핸디 낮은 순
- ✅ 모임별 선택 가능

### **5단계: 시각화**
- ✅ **달력 뷰:** 월별 라운드 기록, 스코어 표시
- ✅ **순위표:** 현재 랭킹, 핸디 포함
- ✅ **라운드 상세:** 홀별 스코어 조회

## 🔄 주요 흐름도

### **① 모임 생성 & 멤버 초대**
```
1️⃣ 운영자: 모임 생성 (모임명, 설명, 랭킹 기준 선택)
2️⃣ 시스템: 모임 코드 + 초대 링크 생성
   예: https://golfrankplatform.vercel.app/join/CLUB-ABC123
3️⃣ 시스템: QR 코드 생성 (같은 링크)
4️⃣ 운영자: 링크/QR 공유 (카톡, 이메일, SNS 등)
5️⃣ 사람들이 링크 클릭:
   ├─ "멤버로 등록" → 영구 참여 (20명 제한)
   └─ "게스트로 참여" → 그 라운드만 (무제한)
6️⃣ 20명 도달 시 → "게스트 참여만 가능" 메시지
```

### **② 라운드 생성 & 스코어 입력**
```
1️⃣ 운영자: 라운드 생성 (날짜, 코스명, Par)
2️⃣ 멤버/게스트: 스코어 입력 선택
   ├─ [수동 입력] → 홀별 타수 1-18홀
   └─ [사진 인식] → 스코어카드 사진 업로드
3️⃣ 사진 업로드 시:
   ├─ Tesseract.js로 이미지에서 숫자 추출
   ├─ 추출된 1-18홀 스코어 프리뷰
   └─ 사용자 검수 후 "확인" → 자동 저장
4️⃣ 라운드 마감 → 핸디 계산 & 순위 업데이트
```

### **③ 핸디 & 랭킹 자동 계산**
```
라운드 마감 직후:
1️⃣ 각 멤버/게스트의 18홀 합계 스코어 확정
2️⃣ 멤버의 경우: 최근 3라운딩 스코어 조회
3️⃣ 핸디 = (3라운드 평균 스코어 - Par)
4️⃣ 모임 설정된 랭킹 기준에 따라 순서 계산:
   ├─ 스코어 기준: 스코어 낮은 순
   └─ 핸디 기준: 스코어 - 핸디 낮은 순
5️⃣ UI 업데이트 (순위표, 달력, 대시보드)
```

## 📸 **OCR 스코어카드 인식 (Tesseract.js)**

### **구현 방식 (프론트에서 클라이언트 인식)**
```javascript
// 1. Tesseract.js CDN 로드
<script src="https://cdn.jsdelivr.net/npm/tesseract.js@5/dist/tesseract.min.js"></script>

// 2. 사진 업로드 → 숫자 인식
async function recognizeScorecard(imageFile) {
  const worker = await Tesseract.createWorker();
  const { data: { text } } = await worker.recognize(imageFile);
  
  // 3. 텍스트에서 1-18 숫자 추출
  const scores = extractNumbers(text); // [4, 5, 3, 4, 5, ...]
  
  // 4. DB에 자동 입력
  await saveScores(roundId, scores);
  
  await worker.terminate();
}

// 5. 숫자 추출 로직 (정규식)
function extractNumbers(text) {
  const matches = text.match(/\d+/g);
  return matches.map(n => parseInt(n)).slice(0, 18);
}
```

### **프로세스**
```
사용자가 스코어카드 사진 업로드
       ↓
Tesseract.js가 이미지에서 텍스트 추출
       ↓
정규식으로 1-18홀 숫자 추출 (ex. 4, 5, 3, ...)
       ↓
추출된 스코어 프리뷰 (사용자 검수)
       ↓
확인 후 DB 자동 저장
```

### **장점**
- ✅ 백엔드 없이 클라이언트에서 처리 (빠름)
- ✅ API 키 불필요 (무료)
- ✅ 오프라인 작동 가능

### **주의**
- ⚠️ 스코어카드 글씨 품질에 따라 정확도 다름 (90-95%)
- ⚠️ 사용자가 수동 검수 후 확인 필요
