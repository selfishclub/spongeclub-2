# 1주차 — 나만의 OS 만들기 🛠️

> 미션을 진행하며 **과정과 결과물**을 기록해주세요. (다 못 채워도 OK, 한 것 위주로!)

## 🎯 미션 1. 내 OS 재료 찾기
> 인터뷰 스킬(아이데이션)로 "내 삶에 필요한 게 뭔지" 찾기

- **과정 (어떻게 찾았나):**
  인터뷰 스킬로 5스텝 인터뷰를 따라갔다. 오늘 하루(아마추어 골프대회 본선 + 롱기스트 여자부 수상 🏌️)에서
  출발해, 반복되며 걸리는 지점을 찾아 내려갔다. 골프 쪽에서는 "몸 관리 + 멘탈"이 매일 신경 쓰이는 결로 나왔고,
  일·일상 쪽으로 각도를 옮기니 "일기·생각·기록이 매일 흩어져 놓침"과 "SNS를 시작하고 싶은데 쉽지 않음"이 떠올랐다.
  세 재료를 나란히 놓고 무게를 재서 **SNS 시작**을 내 OS가 도울 영역으로 골랐다.

- **결과:**
  🟡 내 삶을 돕는 OS · 재료 카드

  ```
  · 영역      : 삶 + 일 (SNS / 콘텐츠 — 뷰티·미용 타겟)
  · 걸리는 지점 : 뷰티·미용 SNS를 시작하고 싶은데, 글·캡션 + 사진·영상 편집
                + 꾸준함이 한꺼번에 막혀 시작을 못 함
  · 지금은    : 소재 맥락(뷰티·미용 + 글로벌 환자)은 잡혔지만,
                막상 0에서 글·캡션을 떠올리는 데서 멈춤
  · OS가 된다면 : 그날 올릴 게시물 소재 + 캡션 초안을 매번 만들어주기
  · 한 문장   : "매일이 콘텐츠로 쌓이는, 하루 1개의 보람찬 날들"
  · 첫 한 걸음 : 클로드 코드로 'SNS 소재·캡션' 채널 하나 —
                주제만 던지면 하루치 초안 1개가 나오게
  ```

- **느낀 점:**
  "딱히 불편한 게 없는데"가 아니라, 막상 하루를 들여다보니 매일 걸리는 게 또렷하게 나왔다.
  나는 간호사 출신으로 지금은 글로벌 환자를 병원과 연결하는 AI 플랫폼 대표로 일하는데,
  이 전문성이 그대로 뷰티·미용 SNS 콘텐츠가 될 수 있다는 걸 인터뷰하면서 확신하게 됐다.

## 🧩 미션 2. 내 OS 기획
> 인터뷰 결과 + 세션 내용(흐민·배짱·키노) 활용해 기획

- **기획 내용:**
  재료 카드를 3단계로 펼쳤다.
  - **1단계 (이번 주):** 주제 하나 던지면 → 하루치 게시물 초안 1개(소재 + 캡션 + 해시태그)가 나오는 채널
  - **2단계:** 플랫폼별 톤 맞추기 (링크드인=전문적 / 인스타·스레드=가볍게 / 유튜브=스크립트형)
  - **3단계:** "이번 주 이 주제 3개 올려요" 리마인드 + 콘텐츠 캘린더처럼 굴리기

  이번 주는 **1단계만** 목표로 한다. 구체적으로는, 클로드 코드에 `SNS-소재공장` 채널을 만들고
  내 맥락(뷰티·미용 + 글로벌 환자 + 간호사 출신 AI 플랫폼 대표)을 미리 학습시킨 상태에서,
  "오늘은 OOO 주제로 인스타용" 하고 한 줄 던지면 게시물 소재 1개 + 캡션 초안 + 해시태그가 나오게 한다.
  목표 플랫폼은 링크드인 · 인스타그램 · 스레드 · 유튜브이고, 하루 1개부터 시작한다.

- **막혔던 점 / 어떻게 풀었나:**
  처음엔 글/캡션·편집·꾸준함이 한꺼번에 막혀 시작 자체를 못 했다.
  OS는 전부 한 번에 하는 게 아니라 "가장 작은 한 조각"부터라는 원칙으로,
  매일의 시작을 막던 '0에서 소재·캡션 떠올리기' 한 가지만 떼어내 1단계로 좁혔다.

## ⚙️ 미션 3. 내 OS 구현
> 실제로 만들어본 것 (클로드코드 '채널' 기능 활용 OK)

- **무엇을 만들었나:**
  흐민님의 이기적 공유회("0 to 100 하지 말고, 가장 불편한 것 하나부터, 내가 이해할 수 있는 간단한 툴로")를
  그대로 적용해, **가장 작은 OS** 하나를 먼저 구현했다.
  미션 2의 SNS 소재공장 대신, 지금 가장 걸리는 **무기력**을 출발점으로 잡아
  **일상 플래너 OS `토닥`** 을 만들었다. (가장 불편한 것 하나부터 — 흐민님 당부 그대로)

- **결과물:** `일상플래너-OS/` 폴더 — 텔레그램·클라우드 없이 **클로드 코드 채팅만으로 오늘 당장** 작동
  - `하루플래너.md` — 플래너 파트너 '토닥'의 규칙·성격 (OS의 뇌). 내 맥락(간호사 출신 AI 플랫폼 대표·골프·SNS·무기력)을 학습시킴
  - `명언집.md` — 아침을 여는 포춘쿠키 명언 풀 (랜덤 1개)
  - `아침브리핑.md` — 매일 아침 브리핑 로직 (명언 + 어젯밤 할일 + 오늘 캘린더)
  - `기록/_오늘_템플릿.md` — 하루 기록 양식
  - `기록/2026-06-30.md` — 첫 날 예시
  - `README.md` — 사용법(아침 "토닥, 오늘 시작" / 저녁 "토닥, 오늘 마무리")

- **동작 방식 (5단계 루프의 가장 작은 버전):**
  아침에 "토닥, 오늘 시작"(또는 "토닥 아침") 한 줄 던짐 → 🥠 포춘쿠키 명언 + 오늘 캘린더 → 컨디션 체크(1~5) →
  오늘 딱 한 가지 → 현실적인 작은 시간표 → 저녁에 "토닥, 오늘 마무리"로 회고 → `기록/`에 그날 파일이 쌓임.
  무기력 대응 규칙: 컨디션 낮은 날은 할 일을 더 잘게 쪼개고 개수를 줄임, 죄책감 금지, 작은 win 짚어주기.
