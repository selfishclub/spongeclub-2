# 1주차 — 나만의 OS 만들기 🛠️

> 미션을 진행하며 **과정과 결과물**을 기록해주세요. (다 못 채워도 OK, 한 것 위주로!)

## 🎯 미션 1. 내 OS 재료 찾기
> 인터뷰 스킬(아이데이션)로 "내 삶에 필요한 게 뭔지" 찾기
- **과정 (어떻게 찾았나):** 셋업데이·1주차 세션을 둘 다 결석해서, 티로 속기본을 클로드에게 정리시켜 먼저 따라잡았다. 그다음 spongeclub-2 레포의 인터뷰 스킬(5스텝)을 클로드 코드로 진행 — 하루 돌아보기 → 짜증 포인트 → 일 밖의 삶 → 무게 재기 → "OS가 된다면" 순서로 답하다 보니 재료가 저절로 좁혀졌다.
- **결과:** 🟡 내 OS 재료 카드
  - **영역**: 일 (대표의 카톡 소통)
  - **걸리는 지점**: 밀린 카톡. 바빠서 못 보고 쌓이고, 몰아서 급하게 답하다 실수가 나온다 (용건만 딱 보내거나, 서두르다 말이 복잡해짐). 클라이언트·직원·개인 방이 혼재돼 있어 더 힘들다.
  - **지금은**: 쌓임 → 틈날 때 몰아서 훑음 → 급한 일부터 무조건 답장 → 빨리 하려다 톤·내용 실수
  - **OS가 된다면**: 밀린 카톡을 보면 ① 도움 될 내용은 정리해주고 ② 답해야 할 건 상대방에 맞는 답변 초안을 준비해준다 (간결한데 따뜻하게, 실수 없이)
  - **한 문장**: "밀린 카톡을 열면, 정리할 건 정리돼 있고 답할 건 상대에 맞는 답변이 준비돼 있다"
  - **첫 한 걸음**: PC 카톡을 클로드가 직접 보고 → 우선순위 정리 + 답장 초안을 뱉어주는 것부터
- **느낀 점:** 질문 몇 개에 답했을 뿐인데 뭘 만들어야 할지가 저절로 정해졌다. 거창한 게 아니라 매일 걸리는 일부터 시작하면 된다는 것.

## 🧩 미션 2. 내 OS 기획
> 인터뷰 결과 + 세션 내용(흐민·배짱·키노) 활용해 기획
- **기획 내용:** 「밀린 카톡 도우미」 — 3단계 로드맵
  - **1단계 (이번 주)**: PC 카카오톡을 화면에 띄워놓고 "카톡 봐줘" 한마디 → 클로드가 화면을 직접 읽고 ① 방별 우선순위 정리(급함/중요/나중) ② 상대별 답장 초안 제안. 복붙 없이 시작하는 반자동.
  - **2단계**: 상대별 말투 프로필 저장 (원장님용/직원용/지인용 톤 구분) → 답장 초안이 자동으로 상대에 맞는 톤으로.
  - **3단계**: 정해진 시간에 밀린 방을 훑어 아침 브리핑까지 (흐민의 설리반처럼 하루의 루틴으로).
- **막혔던 점 / 어떻게 풀었나:** 카카오톡은 개인 대화를 읽을 공식 API가 없다 → "전체 완전 자동"은 애초에 불가능. 복사-붙여넣기 방식은 귀찮아서 지속이 안 될 것 같았다 → 클로드가 PC 카톡 **화면을 직접 보는(캡처)** 방식으로 우회하고, 거창한 자동화 대신 "말하면 봐주는" 반자동 1단계부터 시작하기로 했다. (세션에서 배운 대로: 꼬이면 새로 만들고, 제로 투 백 하지 않기)

## ⚙️ 미션 3. 내 OS 구현
> 실제로 만들어본 것 (클로드코드 '채널' 기능 활용 OK)
- **결과물:** 「밀린 카톡 도우미」 1단계 완성
  - 구성: 규칙 문서(CLAUDE.md) 하나가 전부. "카톡 봐줘" 하면 클로드가 PC 카톡 화면을 직접 보고, 밀린 메시지가 많으면 카톡의 '대화 내보내기' 기능으로 읽어서 — 정보방이면 브리핑, 답장할 방이면 우선순위+답장 초안을 만들어주는 OS.
  - 실제로 밀린 단톡방 하나를 브리핑 한 장으로 정리하는 데까지 성공.
  - 안전장치: 대화 원문·대화 내용은 어디에도 올리지 않음 / 답장 전송은 반드시 내가 최종 확인.
- **링크 / 스크린샷:** (개인 대화가 포함되어 생략합니다)
- **다음 단계(2단계) 예고:** 상대별 말투 프로필(원장님/직원/지인)을 저장해서 답장 초안이 자동으로 맞는 톤으로 나오게.
