# 1주차 — 나만의 OS 만들기 🛠️

> 미션을 진행하며 **과정과 결과물**을 기록해주세요. (다 못 채워도 OK, 한 것 위주로!)

## 🎯 미션 1. 내 OS 재료 찾기
> 인터뷰 스킬(아이데이션)로 "내 삶에 필요한 게 뭔지" 찾기
- **과정 (어떻게 찾았나):** 인터뷰 스킬로 하루를 들여다봤다. 시간을 제일 많이 잡아먹는 게 뭔지 → 어디서 자꾸 걸리는지 → 일 밖에서 놓치는 게 뭔지 순서로 이야기하다 보니 자연스럽게 정리됐다.
- **결과:** 🟡 내 OS 재료 카드
  - **영역**: 일
  - **걸리는 지점**: 결정 못 하고 계속 고민하는 것 — 머릿속에서만 계속 돌림
  - **지금은**: 결론 없이 에너지만 쓰는 잡생각 루프
  - **OS가 된다면**: 고민을 꺼내 적으면 선택지를 정리해주고, 결정 기준을 물어서 답을 좁혀줌
  - **한 문장**: "고민이 결정으로 바뀌는 길이 생긴다"
  - **첫 한 걸음**: "이거 결정 못 하겠어"로 시작하는 대화 흐름 하나 만들기
- **느낀 점:** 딱히 불편한 게 없다고 생각했는데, 얘기하다 보니 매일 잡생각에 에너지를 쓰고 있었다는 걸 알게 됐다. 문제가 없는 게 아니라 너무 익숙해져서 못 보고 있던 것.

## 🧩 미션 2. 내 OS 기획
> 인터뷰 결과 + 세션 내용(흐민·배짱·키노) 활용해 기획
- **기획 내용:**

  **결정 OS** — "이거 결정 못 하겠어"라고 말하면 굴러가는 흐름

  | 단계 | 내용 |
  |------|------|
  | 1단계 (이번 주) | 고민을 한 줄로 적으면 → 선택지 정리 → 결정 기준 질문 → 결론으로 닫아주는 대화 흐름 만들기 |
  | 2단계 | 자주 하는 고민 유형별로 패턴화 (예: 일정 결정, 우선순위, 사람 관련) |
  | 3단계 | 결정 히스토리 쌓기 — 과거에 어떤 기준으로 결정했는지 돌아볼 수 있는 OS |

  이번 주는 **1단계만**: `/결정` 같은 이름으로 클로드 코드 스킬 하나 만들기.

- **막혔던 점 / 어떻게 풀었나:** "내가 뭘 만들어야 할지 모르겠다"가 출발점이었다. 거창한 걸 만들어야 한다는 압박을 내려놓고, 지금 제일 에너지 빼앗기는 것 하나만 골랐더니 방향이 잡혔다.
