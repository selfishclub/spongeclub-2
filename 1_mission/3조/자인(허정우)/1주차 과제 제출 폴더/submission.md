# 1주차 — 나만의 OS 만들기 🛠️

> 미션을 진행하며 **과정과 결과물**을 기록해주세요. (다 못 채워도 OK, 한 것 위주로!)

## 🎯 미션 1. 내 OS 재료 찾기
> 인터뷰 스킬(아이데이션)로 "내 삶에 필요한 게 뭔지" 찾기

- **과정 (어떻게 찾았나):**
  `sponge-and-interview` 스킬로 5스텝 인터뷰를 진행했다. 오늘 하루 되짚기 → 짜증 포인트 → 일 밖의 삶 → 무게 재기 → OS가 된다면, 순서로 가볍게 따라가니 일 쪽(병원 진료 — 반복 설명 유튜브 대체, 자동 차팅, 예진차트 자동화)과 삶 쪽(우선순위 기반 일과표 재조정 비서, 자료 수집, 쇼핑 최저가, 결정장애, 강의 복습 등) 양쪽에서 재료가 쏟아졌다. 병원 쪽 예진차트 아이디어는 이미 구체적으로 그려져 있을 만큼 디테일했지만, "둘 중 하나가 지금 자동으로 굴러가면 어느 쪽이 하루를 더 가볍게 만들까"라는 질문에서는 오히려 삶 쪽(일과표 우선순위)을 골랐다.

- **결과:** 🟡 내 OS 재료 카드

  | 항목 | 내용 |
  |---|---|
  | 영역 | 삶 (개인 일과 관리) |
  | 걸리는 지점 | 구글캘린더에 크고 작은 일을 무작위로 적어놓기만 해서, 우선순위 판단 없이 닥치는 대로 일하게 됨 |
  | 지금은 | 구글캘린더에 일정만 적어놓음 → 중요도·소요시간 고려 없이 방치 → 시간 날 때마다 "뭐부터 하지?" 매번 새로 고민 → 결국 쉽고 사소한 일만 골라 하고, 놓치는 일도 많음 |
  | OS가 된다면 | 일과표를 우선순위대로 짜주고, 시간 됐을 때 뭐 할지 바로 알려줌 + 말로 하면 STT로 받아 적어 기록·분류까지 |
  | 한 문장 | **"고민과 결정으로 날리는 시간 중 두세 시간은 세이브될 것 같다"** |
  | 첫 한 걸음 | 클로드 코드로 '일과표 우선순위 정리' 스킬 하나 — 오늘 할 일을 넣으면 중요도·소요시간 기준으로 정렬해주는 것부터 |

- **느낀 점:**
  처음엔 병원(예진차트 자동화) 아이디어가 훨씬 완성도 높고 상세해서 당연히 그쪽으로 갈 줄 알았다. 그런데 막상 "지금 당장 하루를 더 가볍게 만들 쪽"을 고르는 순간엔 삶 쪽을 선택하게 됐다. 이미 잘 정리된 아이디어가 아니라, 매일 반복되는 사소한 마찰이 진짜 OS 재료라는 걸 체감했다.

## 🧩 미션 2. 내 OS 기획
> 인터뷰 결과 + 세션 내용(흐민·배짱·키노) 활용해 기획

- **기획 내용:** **일과표우선순위-OS** — 구글캘린더에 순서 없이 쌓인 할 일을 붙여넣으면, 고정 일정·마감·소요시간·중요도를 기준으로 자동 정렬해 "오늘의 실행 시간표"로 만들어주는 시스템.

  3단계 로드맵으로 설계하고, 이번 주는 **1단계만** 구현:
  - **1단계 (이번 주)** — 할 일 목록 → 우선순위 정렬 시간표
  - **2단계** — 시간대별 "지금 할 차례예요" 알림(push) + 구글캘린더 직접 연동
  - **3단계 (전체 OS)** — 말로 하면 STT로 "뭘 얼마나 어떻게 했고 결과가 어땠는지" 자동 기록·분류되고, 그 데이터로 다음 날 일과표가 스스로 재조정

- **막혔던 점 / 어떻게 풀었나:**
  처음엔 "우선순위"를 뭘로 정할지 막막했다 — 중요도·긴급도·에너지까지 다 따지려니 규칙이 너무 복잡해졌다. 그래서 판단 기준을 고정 여부 + 4단계 긴급도(지금 안 하면 손해/오늘 안에/이번 주 안에/여유 있음) + 집중·가벼움 정도로 단순화해서 풀었다. 또 "시간표를 빈틈없이 채우는 것"에 욕심내다 멈췄는데, 정작 필요한 건 "지금 뭐부터 하지"에 대한 즉답이라는 걸 깨닫고, 표 아래에 🎯 지금 할 일 한 줄을 반드시 붙이는 규칙으로 범위를 좁혔다.

  실제 구글캘린더·구글태스크 데이터로 검증하면서 한 번 더 막혔다. "고정 여부"를 캘린더에 시간이 찍혀 있는지로만 판단했더니, "세무자료 제출"처럼 실제로는 유동적인데 그냥 입력한 시점(18시) 그대로 고정으로 오판하는 문제가 나왔다. 상대(환자·동료·강사 등)가 관여하는 일인지 아닌지로 기준을 바꿔서 풀었다. 그리고 구글 태스크에 기본 목록만 진행 중인 할 일이 551개나 쌓여 있는 것도 발견했다 — 오늘 하루만이 아니라 이 백로그도 이 OS가 결국 다뤄야 할 문제라는 걸 알게 됐다.

- **어떻게 동작하나:**
  구글캘린더에 순서 없이 적어둔 할 일 목록(진료·미팅 같은 고정 일정 + 차팅·자료 준비·복습·이메일 같은 유동적인 일)을 통째로 붙여넣으면, 고정 일정을 먼저 시간대에 박고 → 남은 시간에 "지금 안 하면 손해"인 일부터 순서대로 배정해서 표로 정리한다. 시간이 모자라 오늘 못 끝낼 것 같은 항목은 미리 ⚠️로 표시해서, 밀릴 걸 당일에 알게 되는 일이 없게 했다.

- **다음 단계(2단계, 예정):** 시간대별 알림(push)과 구글캘린더 직접 연동으로, 지금처럼 매번 붙여넣지 않아도 자동으로 돌아가게.
