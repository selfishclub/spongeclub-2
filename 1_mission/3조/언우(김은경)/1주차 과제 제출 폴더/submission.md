# 1주차 — 나만의 OS 만들기 🛠️

> 미션을 진행하며 **과정과 결과물**을 기록해주세요. (다 못 채워도 OK, 한 것 위주로!)

## 🎯 미션 1. 내 OS 재료 찾기
> 인터뷰 스킬(아이데이션)로 "내 삶에 필요한 게 뭔지" 찾기

- **과정 (어떻게 찾았나):**
  스폰지클럽 인터뷰 스킬로 하루를 5분 되짚어봤다. 5스텝(오늘 하루 → '아 짜증' 포인트 → 일 밖의 나 → 무게 재기 → OS가 된다면 + 한 문장)을 따라가며, 처음엔 막연했던 게 점점 하나로 좁혀졌다. 특히 "일 밖의 나" 질문에서 나온 'AI 공부'와, 일 쪽의 '자료 찾고 정리하기'가 사실은 **같은 문제**였다는 걸 발견했다.

- **결과:** 내 OS 재료 카드
  - 영역: 일 + 삶 (둘 다) — 빠르게 바뀌는 AI를 익혀 업무에 적용하기
  - 걸리는 지점: 최신 AI 툴·솔루션 사용법을 빨리 익혀 정리해 업무에 바로 적용하고 싶은데, **활용처를 못 찾고 '겉핥기 학습'** 으로 끝남
  - 지금은: 유튜브에 의존 → 검색 + 영상 하나하나 보기에 시간 다 씀 → 정작 정리·적용엔 시간 못 씀
  - OS가 된다면: ① 믿을 소스에서 '영상 없이 핵심 요약' → ② 내 업무 '활용처' 연결 → ③ '미니 실습'으로 적용
  - 한 문장: **"영상 대신 요약으로 — 아낀 시간을 진짜 적용에 쓴다."**

- **느낀 점:**
  만들 게 없어서가 아니라 **아이디어가 너무 많아서** 막막했던 거였다. 하루를 같이 되짚어보니 흩어져 있던 게 딱 하나의 진짜 문제로 모였다. "무슨 기능을 만들까"가 아니라 "내 하루의 어디가 자꾸 걸리나"에서 출발하니 방향이 잡혔다.

## 🧩 미션 2. 내 OS 기획
> 인터뷰 결과 + 세션 내용(흐민·배짱·키노) 활용해 기획

- **기획 내용: "적용까지 가는 AI 브리핑 OS"**
  한 방에 다 만들지 않고 **바닥부터 3단계로 쌓는** OS로 기획했다. 세 단계가 과제의 흐름과도 맞는다.
  - **1단계 (이번 주)** — ① 정보 수집·요약: 신뢰 소스를 고정해두고, 새 AI 툴·솔루션 소식을 **영상 없이 핵심만 텍스트로** 요약해 한 곳에 모은다.
  - **2단계** — ② 활용처 연결: 각 소식에 "이건 내 기획·마케팅 업무에 이렇게 쓴다"는 **활용처 한 줄**을 붙인다.
  - **3단계** — ③ 미니 실습: 익힌 것 하나를 **실제 내 업무 예시**에 바로 적용해 '겉핥기'를 막는다.

  **세션에서 얻은 것 (흐민·배짱):**
  - *흐민 세션(5단계 루프: 인풋→저장→연결→제안→발행)* — "허들 없이 자동으로 흐르는 시스템"이라는 개념을 빌렸다. 내 OS도 **"매번 뭐 볼까 찾는 허들"** 을 없애려고 소스를 3~4개로 고정했다.
  - *흐민 텔레그램 봇 연동(Claude Code 채널 기능)* — '엔진'과 '배달'을 분리하는 힌트를 얻었다. 그래서 **요약 로직(스킬=엔진)을 먼저** 만들고, 텔레그램 수신(배달)은 나중에 얹기로 순서를 잡았다.
  - *배짱 온맘(OnMom) 사례* — "작게 시작해 피드백으로 디벨롭"하는 방식을 참고해, 이번 주는 욕심내지 않고 **1단계만** 제대로 만들기로 했다.

- **막혔던 점 / 어떻게 풀었나:**
  - 아이디어가 너무 많아 안 좁혀짐 → **인터뷰 스킬로 하루를 되짚어** 하나로 수렴.
  - 유튜브 의존으로 서칭·학습에 시간 과다 + 겉핥기 → **신뢰 소스 고정 + 영상 없이 텍스트 요약**으로 병목 제거.
  - "채널이냐 스킬이냐" 고민 → **스킬 = 엔진(뭘 하나), 텔레그램 = 배달(어디로 받나)** 로 구분하고 엔진부터.

## ⚙️ 미션 3. 내 OS 구현
> 실제로 만들어본 것 (클로드코드 '채널' 기능 활용 OK)

- **결과물:** 클로드 코드 스킬 **`ai-tool-briefing`** 제작 (`.claude/skills/ai-tool-briefing/SKILL.md`)
  - 고정 소스(TLDR AI · The Rundown AI · GeekNews · 자주 보던 유튜버 1개)를 넣으면
  - 이번 주 새 AI 툴·솔루션을 **영상 없이 핵심 요약** + **기획·마케팅 활용처 한 줄** + **이번 주의 픽**으로 한 장 정리
  - 실제로 돌려서 검증 완료: 이번 주 소식 5개(Canva Grow 2.0, Runway Agent 2.0, LinkedIn 브랜드 킷, ActiveCampaign 2.8, Claude in Advertising)를 **영상 하나 안 보고** 파악. "이번 주 AI 뭐 나왔지"에 들던 30분+ 이 **한 장 읽기**로 줄었다.

- **링크 / 스크린샷:** 스킬 파일 `.claude/skills/ai-tool-briefing/SKILL.md` · 실제 브리핑 결과 스크린샷은 `이미지첨부/` 폴더에 추가 예정
