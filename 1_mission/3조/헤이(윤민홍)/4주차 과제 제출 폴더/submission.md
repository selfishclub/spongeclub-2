# 4주차 – 기획·구현 → 검증 → 기록 ✏️

> **슬쩍 ✦** — 조용히, 그러나 제대로 · AI 카드뉴스 생성기

## 🎯 과제 1. PRD 기획 및 구현
> 주요 가치 · 페르소나 · 소구 방식을 담은 PRD → 구현까지
- **주요 가치:** "SNS가 싫은 게 아니라 나를 드러내는 게 싫은 사람"을 위한 도구. 얼굴·개인 노출 없이 상품/서비스만으로 카드뉴스·영상을 만든다.
- **페르소나:** 소상공인·프리랜서·전문직 1인 사업자 (PC 작업 선호).
- **소구 방식:** 기능 나열이 아니라 단순 UX. 상품명+특징 3줄(또는 키워드 2~3개)만 넣으면 완성.
- **구현한 것 (링크·스크린샷 – 이미지는 `이미지첨부/` 폴더에):** React 웹앱으로 구현.
  - 입력: 상품/일상 2모드, 이미지 0~5장(EXIF 제거·방향보정), 길이·말투 토글
  - 디자인: 스타일 7종 자동 선택, 이미지에서 색상 자동 추출·팔레트, 카드별 줌/크롭·텍스트 즉시 편집
  - 연출: 이모지 7종, 효과 3종(페이드/줌/숨쉬기), 정렬·톤 6종
  - 저장: HD 카드, 영상(피드 4:5 / 릴스 9:16), 무저작권 배경음 6종
  - 👉 [작동 화면 링크 — Claude.ai 공유 링크 붙이기]

## 💬 과제 2. 유저 피드백 받기
> 실제 2명 이상 + 플러스 알파로 가상 페르소나
- **실제 유저 피드백 (2명 이상):** _(아직 미수집 — 실제 후기를 받으면 여기에)_
- **가상 페르소나 피드백:** 4명 검증 완료
  - **A 정영호**(58·괴산·건표고): 사진 없이·국내산 강조 좋음 / 버튼 많고 용어(릴스·캡션·HD) 장벽
  - **B 한소민**(34·성수·비누·캔들): 릴스 감성·색 매칭 좋음 / 잔잔한 배경음 없음·브랜드색이 팔레트에 안 뜸
  - **C 오세권**(46·대구·1인세무): "사진 0장으로도 결과가 나온다"가 결정적 / 무이미지 카드 허전·후킹 톤이 세무와 안 맞음
  - **D 유가은**(29·부산·베이킹): 상품·일상 2모드+경쾌한 배경음 정조준 / 음악 넣으면 인스타 업로드가 어려움
- **피드백에서 알게 된 것:** 반영 완료 — 폰트·크기 통일, 피드 크로스페이드, '국내산' 강제, 잔잔한 배경음(노을) 추가, 팔레트 개선, 무이미지 카드 보강, 전문직용 담백 톤. 남은 리스크: 음악 포함 영상의 인스타 업로드 호환.
