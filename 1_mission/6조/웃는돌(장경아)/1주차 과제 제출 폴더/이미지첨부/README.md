# 이미지첨부

미션 3 '보고-포맷-변환' 스킬의 **실행 화면 캡처**를 이 폴더에 넣어주세요.

## 캡처 방법

1. 클로드 코드에서 스킬 실행:
   ```
   이 인사이트 임원 보고 슬라이드로 바꿔줘
   ```
   그리고 정리해둔 인사이트 텍스트를 붙여넣기.
2. 입력 → 슬라이드 초안 출력까지 나온 화면을 캡처.
3. 캡처 이미지를 이 폴더에 저장 (예: `보고-포맷-변환_실행화면.png`).
4. `submission.md`의 미션 3 '링크 / 스크린샷' 칸에 이미지가 자동 연결됩니다.

## 파일명 예시

- `보고-포맷-변환_실행화면.png`
- `실행결과_슬라이드초안.png`
