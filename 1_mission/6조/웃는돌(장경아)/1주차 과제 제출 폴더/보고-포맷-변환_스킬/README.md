# 보고 포맷 변환기 (report-format-converter)

장경아(웃는돌)의 1주차 OS **1단계** 구현물이에요.
정리해둔 AI 트렌드·전략 인사이트(경쟁사 강약점 / 당사 대응방안 / 시사점)를 넣으면,
**임원 보고용 슬라이드 초안**으로 자동 변환해줍니다.

> 인사이트를 '고민하는 앞부분'은 내가 쥐고, '슬라이드로 옮기는 뒷부분'만 OS가 대신한다.

## 설치 (로컬에서 실제로 쓰기)

이 폴더의 `SKILL.md`를 클로드 코드 스킬 폴더로 복사하면 자동 인식돼요:

```
mkdir -p .claude/skills/보고-포맷-변환
cp "SKILL.md" .claude/skills/보고-포맷-변환/SKILL.md
```

(`.claude/`는 로컬 전용이라 저장소에는 이 제출 사본을 함께 둡니다.)

## 사용법

클로드 코드에 이렇게 말하면 돼요:

```
이 인사이트 임원 보고 슬라이드로 바꿔줘
```
그리고 정리된 인사이트 텍스트를 붙여넣으면 됩니다.

## 출력 구조

`결론 한 줄 → 경쟁구도 → 대응방안(단기/중기) → 실행 요약(Ask)` 순의 슬라이드 초안.
각 장은 `핵심 3점 + So what(시사점)` 구조이고, 입력에 없는 수치는 `[확인 필요]`로 표시합니다.

## 다음 단계 (2·3단계)

- 2단계: 인사이트 추출도 도와주기(자료 넣으면 틀에 맞춰 1차 초안) + 대시보드 포맷
- 3단계: 수집(뉴스·커뮤니티·논문)까지 앞단 자동화 → 수집~보고 전체가 하나의 길로
