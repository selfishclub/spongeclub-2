# 보고 포맷 변환기 (report-format-converter)

- 장경아(웃는돌)의 1주차 OS **1단계** 구현물
- 입력: 정리된 AI 트렌드·전략 인사이트 (경쟁사 강약점 / 당사 대응방안 / 시사점)
- 출력: **임원 보고용 슬라이드 초안** 자동 변환
- 역할 분담: '고민하는 앞부분'은 사람, '슬라이드로 옮기는 뒷부분'은 OS

## 설치 (로컬에서 실제로 쓰기)

- `SKILL.md`를 클로드 코드 스킬 폴더로 복사 → 자동 인식

```
mkdir -p .claude/skills/보고-포맷-변환
cp "SKILL.md" .claude/skills/보고-포맷-변환/SKILL.md
```

- 참고: `.claude/`는 로컬 전용 → 저장소에는 이 제출 사본을 함께 보관

## 사용법

- 클로드 코드에 입력:

```
이 인사이트 임원 보고 슬라이드로 바꿔줘
```

- 이어서 정리된 인사이트 텍스트 붙여넣기

## 출력 구조

- 슬라이드 순서: `결론 한 줄 → 경쟁구도 → 대응방안(단기/중기) → 실행 요약(Ask)`
- 각 장 구성: `핵심 3점 + So what(시사점)`
- 미확인 수치: `[확인 필요]`로 표시 (지어내지 않음)

## 다음 단계 (2·3단계)

- 2단계: 인사이트 추출 지원 (자료 → 틀에 맞춘 1차 초안) + 대시보드 포맷
- 3단계: 수집(뉴스·커뮤니티·논문) 앞단 자동화 → 수집~보고 전체 연결
