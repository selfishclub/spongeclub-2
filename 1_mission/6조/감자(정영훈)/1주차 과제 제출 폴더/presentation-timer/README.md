# 🧽 스폰지 타이머

발표는 물론 **일상에서도** 쓰는 크롬 타이머. 보고 있는 웹페이지 위에 **드래그되는 플로팅 타이머/스톱워치**를 띄운다. 스폰지클럽 2기 1주차 과제로 제작.

## ✨ 기능

- **두 모드** — ⏱ 카운트다운 타이머 + ⏲ 스톱워치(0부터 경과 시간 재기)
- **플로팅 오버레이** — 페이지 위에 떠 있고, 헤더를 잡고 원하는 위치로 드래그
- **단축키** — 팝업을 거치지 않고 바로 띄우고 시작/정지 (아래 참고)
- **색상 경고**(타이머) — 초록 → 노랑(30초) → 빨강(10초) → 초과 시 깜빡임
- **소리 알림**(타이머) — 30초·10초·종료 시점 비프음 (🔔/🔕 토글)
- **초과 시간**(타이머) — 0이 되면 멈추지 않고 `+00:15`처럼 넘긴 시간 표시 / `+1:00` 연장
- **프리셋** — 1·3·5·7·10분 + 뽀모도로 25분·휴식 5분 + 분·초 직접 입력
- **이어보기** — 종료시각·시작시각(절대시각) 기준이라 새로고침 후 다시 띄워도 이어짐

## 📦 설치 (개발자 모드)

1. 크롬 주소창에 `chrome://extensions` 입력
2. 우측 상단 **개발자 모드** 켜기
3. **압축해제된 확장 프로그램을 로드합니다** 클릭 → 이 폴더 선택
4. 툴바에 🧽 스톱워치 아이콘이 보이면 완료 (안 보이면 퍼즐 아이콘 → 📌 고정)

> 한글·공백이 들어간 경로보다 **영문 경로**(예: `~/sponge-timer`)에 두고 로드하는 걸 권장. 개발·압축·공유 시 안정적이다.

## ▶ 사용법

### 타이머 (카운트다운)
1. 🧽 아이콘 클릭 → **타이머** 탭 → 시간 설정 → **타이머 띄우기 ▶**
2. 오버레이의 **▶ 시작** 으로 카운트다운 시작

### 스톱워치 (경과 시간)
1. 🧽 아이콘 클릭 → **스톱워치** 탭 → **스톱워치 띄우기 ▶**
2. **▶ 시작** 누르면 0부터 카운트업. 공부·요리·운동 시간 재기에 좋다.

### 단축키 (팝업 없이 바로)
| 동작 | macOS | Windows/Linux |
|------|-------|---------------|
| 타이머 띄우기/시작·정지 | `⌘ ⇧ Y` | `Ctrl Shift Y` |
| 스톱워치 띄우기/시작·정지 | `⌘ ⇧ U` | `Ctrl Shift U` |

- 누르면 현재 탭에 바로 뜨고 시작, 다시 누르면 정지/시작이 토글된다.
- 단축키 변경: `chrome://extensions/shortcuts`
- 단축키로 시작했을 땐 소리가 잠겨 있을 수 있는데, 오버레이를 **한 번 클릭**하면 켜진다(브라우저 자동재생 정책).

## 🔐 권한

| 권한 | 쓰는 이유 |
|------|-----------|
| `activeTab` + `scripting` | 아이콘/단축키를 쓴 **그 탭에만** 오버레이를 주입 |
| `storage` | 모드·설정 시간·소리·위치·진행 상태 저장(이어보기) |
| `background`(service worker) | 단축키 입력을 받아 현재 탭에 띄우기 |

> `<all_urls>` 같은 광범위 권한을 쓰지 않아 "모든 웹사이트 데이터 읽기" 경고가 뜨지 않는다.

## ⚠️ 알아둘 점

- 브라우저 **탭 안에서** 동작한다. 데스크탑 키노트·PPT 앱 화면 위에는 뜨지 않는다.
- `chrome://`, 크롬 웹스토어, 새 탭 페이지에는 (브라우저 정책상) 주입할 수 없다.

## 🗂 파일 구조

```
├── manifest.json     # 익스텐션 정의 (MV3) · 단축키 commands

├── popup.html/css/js # 모드 선택 + 시간 설정 런처
├── overlay.css       # 플로팅 타이머/스톱워치 스타일
├── content.js        # 타이머·스톱워치·드래그·소리
└── icons/            # 스폰지 스톱워치 아이콘 (16/48/128)
```
