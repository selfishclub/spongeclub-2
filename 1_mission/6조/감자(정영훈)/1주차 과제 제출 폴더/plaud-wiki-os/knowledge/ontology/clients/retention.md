# 고객 유지 & 업셀

> 관련 원칙: [[mentor-k-sales]], [[mentor-k-people]]

> n건 → n건으로 가는 길의 절반은 신규 영업이고, 나머지 절반은 기존 고객 유지.

## 이탈 방지 전략

### 자동 리포트
- 주간: GBP 랭킹 변동, 리뷰 현황
- 월간: 종합 성과 리포트 (gbp-local PPT)
- 고객이 "효과가 보인다"고 느껴야 유지됨

### 이탈 조기 경보
- 계약 만료 30일 전 리뉴얼 안내 (자동화 예정)
- 랭킹 하락 시 즉시 원인 분석 + 대응 보고
- 리뷰 0건 상태 지속 시 경고

## 업셀 트리거

| 트리거 | 제안 |
|--------|------|
| BASIC 3개월 후 랭킹 상승 입증 | STANDARD 제안 (4언어 + 광고) |
| 외국인 리뷰 증가 | 추가 언어 애드온 (마진 n%) |
| 고객이 영상 언급 | 숏폼 추가 또는 PREMIUM |
| 경쟁사 광고 발견 | 검색광고 대행 제안 |

## 유지율 추적

- 목표: 월 이탈률 n% 이하 (계약 n건 기준 월 n건 이하)
- 현재 활성 계약: n건 (clients.json 기준 n업장 등록, 활성 계약은 별도 확인 필요)
- 측정 시작: (매월 1일 집계)

## 위험 고객 조기 식별 기준

| 위험 신호 | 확인 방법 | 대응 |
|----------|----------|------|
| 랭킹 하락 지속 (2주+) | gbp-local 주간 스캔 | 원인 분석 + 대응 보고 |
| 고객 응답 지연 (7일+) | 카톡/이메일 | 직접 전화 팔로업 |
| 계약 만료 30일 이내 | pluuug (미연동) → 수동 체크 | 리뉴얼 안내 + 성과 리포트 |
| 리뷰 0건 2주 지속 | gbp-local 리뷰 동기화 | 리뷰 캠페인 제안 |
