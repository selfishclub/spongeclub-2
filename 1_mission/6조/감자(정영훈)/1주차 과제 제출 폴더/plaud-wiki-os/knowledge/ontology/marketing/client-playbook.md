# 고객 마케팅 실행 플레이북

> 관련 원칙: [[mentor-k-marketing]], [[mentor-k-sales]]

> 병원 고객에게 제공하는 마케팅 서비스의 실행 매뉴얼.

## GBP SEO (핵심 서비스)

### 초기 최적화 체크리스트
- [ ] 비즈니스 정보 완전성 (주소, 전화, 영업시간, 카테고리)
- [ ] 프로필 사진 최적화
- [ ] 서비스 목록 등록
- [ ] 다국어 설정 (계약 언어 수만큼)
- [ ] 초기 게시글 작성
- [ ] 리뷰 응답 전략 수립

### 주간 작업
- GBP 게시글 1~2회
- 신규 리뷰 모니터링 및 응답
- 랭킹 변동 체크 (gbp-local)

### 월간 작업
- 월간 리포트 작성/발송
- 키워드 랭킹 추이 분석
- 경쟁사 동향 체크
- 개선 제안

## 구글 검색광고 (STANDARD 이상)

- 협력업체7에 완전 일임
- 월간 성과 리포트 확인 후 고객에게 전달
- 광고비 조정 제안은 데이터 기반으로

## 영상 콘텐츠 (STANDARD: 숏폼, PREMIUM: 풀패키지)

- G엔터/협력업체5 제작
- 기획 방향 → 병원 특성 + 타겟 환자 기반
- 완성물 QC → 고객 확인 → 게시
