# 온보딩 가이드

> 관련 원칙: [[mentor-k-people]]

> 새 작업자/영업대행사가 합류할 때, AI가 이 문서를 기반으로 교육할 수 있다.

## 작업자 온보딩

### 우리 회사가 하는 일
- 병원(의료기관)의 Google Maps 상위노출을 도와주는 마케팅 대행사
- 핵심 서비스: GBP(Google Business Profile) SEO
- 외국인 환자 유치가 주요 가치 제안

### 작업 범위
- GBP 프로필 최적화 (업장 정보, 사진, 게시글)
- 다국어 SEO (한/영/중/일)
- 주간/월간 리포트 작성 보조

### 보수
- 업장당 n만/월 (프리랜서, 3.3% 원천징수)
- 언어 수 무관, 업장 단위

### 필요 도구
- Google Business Profile 접근 권한
- Canva (게시글 이미지 제작)
- 구글 스프레드시트 (작업 진행 추적)

### GBP 작업 매뉴얼 (핵심 체크리스트)

#### 초기 세팅 (계약 후 1주)
- [ ] GBP 접근 권한 받기 (관리자 또는 소유자)
- [ ] 업장 기본정보 확인/수정 (주소, 전화, 영업시간, 카테고리)
- [ ] 한국어 설명 최적화 (키워드 포함)
- [ ] 영어/중국어/일본어 설명 작성 (계약 언어 수에 따라)
- [ ] 사진 최소 10장 업로드 (외관, 내부, 시술 전후, 의료진)
- [ ] FAQ 10개 이상 등록

#### 주간 루틴
- [ ] 다국어 게시글 2~3회 (한/영/중/일 각 1회)
- [ ] 신규 리뷰 확인 + 답변 작성 (24시간 이내)
- [ ] 위험 리뷰 발견 시 즉시 보고

#### 월간 체크
- [ ] 사진 추가 (최소 5장/월)
- [ ] 게시글 성과 확인
- [ ] 랭킹 변동 확인 (gbp-local 리포트 기반)

## 영업대행사 온보딩

### 상품 이해
- [[pricing|pricing.md]] 필독
- BASIC/STANDARD/PREMIUM 3단 구조
- BASIC으로 진입 → 성과 증명 → 업셀이 핵심 루트

### 영업 도구
- 무료 GBP 진단 리포트 (자동 생성)
- 이북 "구글 지도 상위노출 가이드"
- 성공 사례 (clients/case-studies.md)
