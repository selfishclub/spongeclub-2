사용자는 번역투·AI투 표현에 강한 거부감이 있다. 모든 외부 발송 문서·내부 메모·응답에 적용.

**Why:** AI가 만든 한국어 문서는 어색한 직역체·한자어 남발·외래어 산발로 신뢰도가 떨어진다. 업주에게 보내는 리포트에서 이런 표현이 나오면 즉시 어설픈 외주 인식. 사용자가 직접 잡아낸 사례에서 발화 — "환장하겠네", "어처구니가 없네"

**금지 표현 → 대체:**
| ❌ 금지 | ✅ 자연 |
|---|---|
| 박힌 X / X가 박혀있다 | X가 들어간 / 비즈니스명에 X 있음 |
| X 자산 / 동선 자산 | X (그냥 "자산" 빼기) |
| 직격이 약합니다 | 효과가 약합니다 |
| 광맥 / 미개발 광맥 | 기회 / 아직 활용 안 한 기회 |
| 장벽 | 걸림돌 |
| 정합 / 정합성 | 맞음 / 어울림 |
| 캡처 / 잡아내다 | 확보 |
| 베이스라인 | 기준점 |
| Blue Ocean | 비어있는 자리 |
| USP / 셀링포인트 | 강점 |
| 광역 | 넓은 범위 |
| 자생 유입 | 알아서 들어오는 검색 |
| ROI | 효율 |
| 격상 | 올림 |
| 시딩 | 심기 |
| 헤드라인 점령 | 상단 차지 |
| 트리오 | (그냥 풀어 쓰기) |
| 음역 | 발음 표기 |
| 정조준 | 정확히 겨냥 |
| 백오피스 유입 | 구글 지도 검색량 |
| Pack 진입 | 상위 3위 안 노출 |
| SoLV 100% | 측정 지점 모두 상위 3위 안 |
| AGR 1.00 | 평균 순위 1위 |
| "1위" "만점" 단정 | "0.1마일 반경 측정에서 상위 3위 안" 측정 조건 명시 |

**가독성 원칙:**
- 한 문장 1~2줄 (긴 호흡 금지)
- 능동태 우선
- 결론 먼저, 근거 다음
- 표 남발 금지 — 문장 흐름이 자연스러우면 표 빼라
- 한자어·외래어는 풀어쓸 수 있으면 풀어쓰기
- 영어 단어 한국어로 대체 가능하면 대체

**도메인 용어 예외:**
- "Pack 진입" "GBP" "SoLV" "AGR" 같은 GBP 측정 도구 용어는 그대로 (단, 첫 사용 시 1줄 정의)

**적용 범위:** 모든 외부 리포트 + 내부 메모 + 사용자 응답
