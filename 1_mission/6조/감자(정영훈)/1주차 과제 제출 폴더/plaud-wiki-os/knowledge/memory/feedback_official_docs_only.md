시스템 설계/리팩토링 시 Anthropic 공식 문서(docs.anthropic.com, GitHub)만 참고한다.
유튜브/블로그의 overhyped 방법론은 배제.

**Why:** 사용자가 "너무 많은 overhyped 방법론이 지시를 누더기로 만든다"고 명시적으로 경고.
**How to apply:** 새 기능/패턴 도입 시 공식 문서 출처 명시. 월 1회 docs changelog 체크하여 온톨로지 operations/tools.md에 반영.
