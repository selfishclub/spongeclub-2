## 비전
"대한민국을 방문하는 모든 외국인 환자의 필수 관문이자, 데이터 기반 글로벌 헬스케어의 표준"

## Origin (사업 출발 배경)
1. **2023년부터 우리 회사의 검증** — 다수 한국 병원의 로컬 SEO를 운영하며 구글 지도 상위 노출의 매출 효과를 입증
   - 다국어 GMB 타이틀 스터핑 (한국어 + 영어 + 일본어 + 중국어 + 태국어) — project_company_multilang_seo_moat.md
   - E의원·C의원 동대문·C의원 신사 GMB 검색어 데이터로 외국인 직접 검색 demand 실증
2. **병원의 공통 요청** — "광고비 따로 쓸 필요 없이, 구글 지도에서 1위 만들어 주고 성과로 정산하자, n%는 무조건 줄 수 있다"는 originating 병원 1곳의 자발적 제안 → 후속 25개 병원 모집
3. **전통 외국인 에이전시 한계** (검증된 페인):
   - 수수료 n%
   - 광고 채널 통제 불가
   - 통역사 비위행위 (에이전시 A → B redirect 부도 처리 사례)
   - 외국인 환자도 에이전시 신뢰 안 함
4. **광고 신뢰도 하락 + AEO·discovery 부상** — 한국 안에서 외국인 환자 입국 후 GMB 워크인 증가, 에이전시·SNS·paid ads 유입 감소 (병원 보고)
- → 서비스M은 '성과제 + 디지털 자산 구축' 모델로 출발

## Demand Evidence Update (2026-05-08)
- 10/25 병원 본 계약 시 n% 수수료 구두 약속
- E의원 returning customer (효과 stick 검증)
- 자세한 evidence: project_service_m.md "Demand Evidence" 섹션

## 사업 목적 (3축)
1. **수익 창출** — 의료관광 연계 직접 유치 수수료 + 병원 대상 마케팅 솔루션
2. **자산 형성** — 광고 소모가 아닌 병원의 구글 맵 권한·데이터를 '디지털 자산'으로 구축
3. **글로벌 경쟁력** — 구글 내비게이션·예약 시스템 선점, 글로벌 헬스케어 데이터 표준 확보

## 3-Tier 서비스 플랜

| 구분 | 베이직 | 스탠다드 | 프리미엄 |
|------|--------|----------|----------|
| 병원 니즈 | "리스크 없이 시작" | "매출 본격 성장" | "전국 1등" |
| 광고 예산 | 0원 | 월 n만 원 이상 권장 | 월 n만 원 이상 권장 |
| 광고 수수료 | 없음 | 광고비의 n% | 광고비의 n% |
| 예약 수수료 | 환자 결제액의 **n%** | 환자 결제액의 **n%** | 환자 결제액의 **n%** |
| 언어 범위 | 2개 국어 선택 최적화 | 4개 국어 전체 최적화 | 4개 국어 + 전략 국가 추가 |
| 광고 소재 | 해당 없음 | 미포함 (기존 활용) | 전담 제작 (기획+제작) |
| 관리 수준 | 상위노출 무료 셋팅 (1회) | 매주 업데이트 + 리뷰 관리 + 지도 검색 최적화 대응 | 최상위 노출 유지 + 광고 운영 일임 |
| 상담 지원 | 예약 확정 + 기본 안내 | 채팅 전문 상담원 다국어 응대 | 1:1 전담 코디네이터 케어 |
| 랜딩페이지 | 전용 페이지 | 전용 페이지 + GA4 추적 | 실시간 방문 + VIP 매칭 |

## 수익 모델 (3축)
1. **플랫폼 예약 수수료**: 베이직 n% / 스탠다드·프리미엄 n%
2. **구글 광고 관리 수수료**: 광고비의 n% (스탠다드 이상)
3. **디지털 자산 구축**: 구글 맵 최적화로 병원의 영구 마케팅 자산 형성

## 시장 차별성
- **전통 에이전시**: 수수료 줄 때만 환자 송출, 계약 종료 시 효과 소멸 (단순 비용)
- **서비스M**: 구축한 구글 지도 순위·SEO는 **병원 소유**. 광고 멈춰도 상위 노출 지속 (마케팅 자산)

## 조직 구성 (3팀)
- **기술운영팀**: 구글 예약 시스템 연동, 로컬 SEO 최적화, GA4 데이터 분석
- **글로벌마케팅팀**: 국가별 구글 Ads 운영, 로컬가이드 리뷰, 광고 소재 기획·제작
- **전략영업팀**: 의료기관 네트워크 확장, 외국인 전담 상담 코디네이터 관리

## 실행 프로세스 (4단계)
1. **진단** — 구글 맵 노출 상태 + 경쟁 병원 분석
2. **셋팅** — 플랜별 언어 최적화 + 광고 소재 기획 (프리미엄)
3. **운영** — 정기 업데이트 + 다국어 상담 지원
4. **성과** — 데이터 분석 리포트 + 효율 최적화

## 핵심 메시지
- "구글 공식 예약 파트너 기반 글로벌 의료 유치 및 마케팅 플랫폼"
- "디지털 자산을 만들어 드립니다" (vs 단순 비용)
- 영문 표기 규칙 고정 (사업계획서 원본 표기 그대로 사용, 임의 변경 금지)
