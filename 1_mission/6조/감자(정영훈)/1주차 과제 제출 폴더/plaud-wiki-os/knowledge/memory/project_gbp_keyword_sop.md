# GBP 키워드 리서치 SOP — 6단계

**문서 위치:** `gbp-dashboard/SOP_키워드리서치.md`
**스크립트:** `company-pipeline/scripts/autocomplete_check.py`
**체크리스트 템플릿:** `[우리 회사] 병의원 구글 지도 체크리스트.docx`

## 6단계 워크플로우

0. **체크리스트 회신 수집** — 업주 작성 7섹션 docx. 핵심은 섹션 3(희망 키워드, 외국인 응대 역량), 섹션 6(상호·주소·영업시간). 타겟 언어는 "외국인 응대 역량"으로 결정
1. **유입 키워드 캡처 + 업주 희망 대조** — GBP 백오피스 검색어 최근 5개월, 언어별 분류. 업주 희망 5개를 교집합/차집합으로 나눠 Step 3 검증 대상화
2. **리뷰 실증** — Review Audit PDF에서 타겟 언어 리뷰 대표 2~3개 + 자연 표현 추출
3. **자동완성 검증** — `autocomplete_check.py --lang ja --region jp` 로 **업주 희망 5개 + 단독/조합** 패턴 모두 확인
4. **최종 선정** — 지역 1~2 + 카테고리 1 + 시그니처 메뉴 1, 총 3~5단어
5. **업주 제안서** — HTML+MD 2벌, 9개 섹션 필수 (✨NEW: 업주 희망 키워드 검증 결과 섹션 포함)

## 핵심 원칙

- **Ubersuggest 볼륨 신뢰 금지** (부정확) — 구글 자동완성 존재 여부로 판단
- **3중 교차 검증 필수**: 유입 데이터 + 리뷰 증거 + 자동완성 **모두** 긍정일 때만 채택
- **조합 > 단독**: 단독 자동완성이 강해도 일본 국내 업체 검색 노이즈 가능, 지역+X 조합이 실사용 검색
- **정체성 > 검색량**: 3~5단어로 짧게 유지, 길어지면 희석

## D매장 레퍼런스 (2026-04-21)

- 일본어 유입 月 500+ 있는데 GBP 이름에 일본어 0 → 결정적 기회
