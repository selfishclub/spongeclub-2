---
name: 터미널 상태 보고
description: 사용자가 터미널/프로세스 구동 상태를 물어보면 현재 실행 중인 프로세스, 서비스, 백그라운드 작업 등을 조사해서 보고
type: feedback
---

사용자가 "어떤 터미널이 돌아가고 있어?" 등 터미널/프로세스 상태를 물어보면, 실제로 실행 중인 프로세스를 확인해서 보고해라.

**Why:** 사용자가 여러 서비스(텔레그램 봇, 로컬 서버 등)를 동시에 운영하므로 현재 상태 파악이 중요함.

**How to apply:** `ps`, `lsof`, `pm2 list` 등으로 실행 중인 프로세스를 확인하고, 어떤 서비스가 어떤 포트/상태로 돌고 있는지 요약 보고.
