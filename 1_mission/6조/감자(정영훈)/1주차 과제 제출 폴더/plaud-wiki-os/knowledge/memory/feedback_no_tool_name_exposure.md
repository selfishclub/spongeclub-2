거래처에 발송하는 모든 자료에 우리 내부 측정 도구 이름 노출 금지.

**Why:** 도구 이름은 우리 작업 도구일 뿐 거래처에게 의미 없음. 노출하면 어설픈 외주 인식. 거래처는 결과만 본다.

**금지 표현 → 대체:**
- "LocalFalcon 측정 결과" → "측정 결과" 또는 "매장 0.1마일 반경 측정 결과"
- "DataForSEO로 확인" → "확인 결과"
- "autocomplete_check.py로 검증" → "자동완성 검증 결과"
- "GBP 백오피스 데이터" → "구글 지도 검색량" (이미 메모리 [번역투/AI투] 적용)

**적용 범위:** 카톡 복붙 자료 + 외부 리포트 + 견적서 + 모든 거래처 발송 문서. 내부 메모는 그대로 사용 가능.
