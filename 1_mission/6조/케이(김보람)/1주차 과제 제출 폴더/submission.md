# 1주차 — 나만의 OS 만들기 🛠️

> 미션을 진행하며 **과정과 결과물**을 기록해주세요. (다 못 채워도 OK, 한 것 위주로!)

## 🎯 미션 1. 내 OS 재료 찾기
> 인터뷰 스킬(아이데이션)로 "내 삶에 필요한 게 뭔지" 찾기

- **과정 (어떻게 찾았나):** Claude Code의 인터뷰 스킬(sponge-and-interview)로 5단계 아이데이션 진행. "정신 에너지를 갉아먹는 게 뭔지"부터 시작해서, 마찰점 → 삶의 목표 → 현재 상태 → OS가 해줬으면 하는 것 순서로 좁혀갔다.

- **결과:**

  🟡 **내 삶을 돕는 OS · 재료 카드**

  | | |
  |---|---|
  | **영역** | 삶 — 개인 재무 + 미래 준비 |
  | **걸리는 지점** | 통장·카드 내역이 흩어져 있어 수입·고정지출·변동지출을 한눈에 못 봄 → 여유자금이 얼마인지 모름 → IRP·연금·ISA·투자 계획이 전부 막힘 |
  | **지금은** | 월급 들어오고 자동이체 나간 뒤 남은 걸로 파악. 다음 월급 직전에야 "이번 달 이만큼 남았구나"를 알게 됨 — 계획이 아니라 결과로 파악하는 구조 |
  | **OS가 된다면** | 수입 / 고정지출 / 변동지출 세 칸으로 정리 → 여유자금과 투자 가능 금액이 한눈에 보임 |
  | **한 문장** | *"미래에 대한 자신감이 생긴다. 하루하루 불안하고 불투명하게 사는 게 아니라."* |
  | **첫 한 걸음** | 이번 달 통장·카드 내역을 한 곳에 모아 세 항목으로 분류해보기 |

- **느낀 점:** 인터뷰 전에는 그냥 "돈 관리가 안 된다"는 막연한 답답함이었다. 질문을 따라가다 보니 핵심이 "여유자금을 모르는 것"이고, 그게 IRP·연금·투자 계획 전부를 막고 있다는 걸 처음으로 또렷하게 봤다. 그 밑에는 "아이가 생겼을 때 회사 말고 가족 곁에 있고 싶다"는 목표가 있었는데, 인터뷰를 통해 그게 지금 돈 관리와 직접 연결된다는 걸 다시 확인했다. 막연한 불안이 구체적인 문제로 좁혀지니까, 뭘 만들어야 할지가 바로 보였다.

## 🧩 미션 2. 내 OS 기획
> 인터뷰 결과 + 세션 내용(흐민·배짱·키노) 활용해 기획

- **기획 내용:**

  3단계 구조로 기획했다.

  - **1단계** (이번 주): 급여 - 고정지출 입력 → 여유자금(잔여) 파악
  - **2단계**: 변동지출(카드 내역) 연동 → 실제 지출 흐름 파악
  - **3단계**: 부업 수익·공모주·투자 수익 연동 → 전체 자산 흐름 통합

  핵심 원칙은 하나: **"현재를 먼저 알아야 계획을 할 수 있다."** 지도를 그리기 전에 내가 어디 있는지부터 찍는 것.

- **막혔던 점 / 어떻게 풀었나:** 처음엔 "모든 걸 한번에" 만들려고 하니 너무 복잡해서 시작을 못 했다. 인터뷰에서 "3단계 중 1단계만"으로 좁히고 나서야 바로 실행할 수 있었다. 이미 고정지출은 어느 정도 정리해둔 게 있어서 "급여 - 고정지출"부터 입력하는 걸 시작점으로 잡았다.
