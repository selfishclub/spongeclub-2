## 🎯 미션 1. 내 삶을 돕는 OS 최종 완성
> 지금까지 공유하며 받은 **피드백을 반영해 최종 완성**!

**내 OS: PAZX** — "생각이 만나는 순간, 파직." 1주차 텔레그램 '영감 수집함' → 2주차 웹서비스 → **3주차엔 하나의 '우주 탐사' 세계관으로 재구성**하고, 아이디어가 실제 **제안서 결과물**까지 착지하는 파이프라인을 완성했다.

- **라이브:** https://pazx.vercel.app
- **코드:** https://github.com/Koo-bon/clapbam

- **완성한 것 (무엇을):**
  - **세계관으로 묶기** — 흩어진 기능을 "아이디어가 여행하는 항로" 한 줄로 다시 꿰었다. 하단 탭이 곧 단계다: **홈 → 아이디어 잇기 → 제안서 → AI팀 → 자료함.**
  - **아이디어 잇기(컨셉 성단)** — 배경 아무 곳이나 눌러 아이디어 별을 놓고(직접 입력 / 내 아카이브 / 무작위 웹), 드래그로 선을 이어 **3개가 이어지면 '별자리 = 컨셉'**이 된다. 별자리가 완성되면 AI가 그 조합에서 **하나의 컨셉(이름·정의·톤·키워드)**을 자동 도출.
  - **제안서 항로** — 컨셉을 "출항"시키면 역할별 AI 크루가 순서대로 일한다: **정찰(인사이트) → 훅카피(슬로건) → 키비주얼(실사 이미지) → 덱 조립 → 최종 검수.** 결과로 슬로건 + 키비주얼이 얹힌 웹 덱이 나오고 **PDF / 편집가능 PPTX로 내보내기**까지 된다.
  - **첫 사용자 온보딩** — 인트로 애니메이션 → **기능별 가이드 투어**(탭마다 카드 한 장씩, 그 탭으로 자동 전환하며 설명) → 컨셉보드 가이드. 가이드 카드는 **리퀴드 글래스**로 만들고, 카드 안에 그 기능이 작동하는 장면(단어 충돌→점화, 별 잇기, 우주선 항로 이동)을 **루프 애니메이션**으로 넣었다. 로그인 화면도 블랙 + 내 X 로고로 정리.

- **피드백 반영한 점:** ("처음 써본 사람" 관점으로 전 구간을 직접 돌며 잡은 것들)
  - **키비주얼이 'AI 티' 났다** — 상황·컨셉이 안 읽히고 콜라 물이 파란색으로 나오던 문제. 이미지 규칙을 **상황 명료성 · 제품 원화 충실(색 왜곡 금지) · 은유는 아트디렉션으로 · 실사 에디토리얼 톤**으로 다시 써서 "네온 밤거리, 손에 든 콜라" 같은 실사 광고급으로 끌어올림.
  - **별자리 인터랙션** — "4번째가 자꾸 자동으로 붙는다 / 선이 원과 어긋난다 / 컨셉 카드 잡으려다 바탕 누르면 꺼진다"를 직접 써보며 하나씩 수정(자동 흡수 제거·드래그 연결·좌표 정렬·클릭 의도 분리).
  - **아카이브 정리** — 깨진 더미 이미지(한글·공백 미인코딩)를 걷어내고, "내 자료가 다 안 들어온다"는 시드 병합 버그를 수정.
  - **덱 내보내기 타이밍** — 키비주얼 합성(~40초) 중 버튼이 눌려 **표지 이미지 빠진 덱**이 나갈 수 있던 것을, 합성 중 버튼 잠금 + "합성 중" 표시로 막음.

- **결과물 (링크·스크린샷):**
  - 동작하는 라이브: https://pazx.vercel.app (로그인 → 아이디어 던지기 → 별자리로 컨셉 완성 → 제안서 항로 출항 → 슬로건·키비주얼·덱)
  - 코드 저장소: https://github.com/Koo-bon/clapbam

- **알게 된 인사이트:** **세계관이 곧 UX가 된다, 그리고 첫 사용자에겐 '설명'보다 '작동하는 장면'이 통한다.** 흩어진 기능을 '별→별자리→항로' 한 줄로 묶으니 화면마다 존재 이유가 저절로 설명됐고, 온보딩도 글로 나열할 때보다 그 기능이 움직이는 미니 데모를 보여줄 때 훨씬 빨리 이해됐다. 그리고 AI 이미지는 은유를 "합성"으로 우기면 티가 나고 **아트디렉션으로 풀어야** 광고급이 된다는 걸 확실히 배웠다.
