# 2주차 — 내 OS 구현하기 🚀

## 🎯 미션 1. 내 OS 만들기

**✅ 선택:** 내 삶을 돕는 OS (일·업무 OS)

> 1주차에서 발견·기획한 **길잡이**를 이번 주에 실제로 만들기 시작했다.
> 방향은 "내 업무 자동화"가 아니라 **"팀의 공통 업무를 자동으로 굴리는 것".**

### 📐 기획 — 무엇을, 왜, 어떻게

대상은 "AI를 못하는 사람"이 아니라, **AI를 자기 업무(특히 사내 시스템 기반)에 어떻게 접목할지 모르는 사람**. AI 자체는 쓰지만 — "똑같이 물어도 안 되는" 이유는 각자의 AI에 **팀 공통 맥락(데이터가 어디 있는지·팀 규칙)** 이 없어서다. 질문을 조금 바꾸거나 사내 지식만 살짝 얹으면 바로 작동한다.

그래서 길잡이는 **팀 범용 스킬** 하나로, 세팅은 **활동수집 한 걸음**. 두 축으로 움직인다:
1. **(개인)** 내 반복 요청을 감지 → *"이거 매주 반복하시네요, 스킬로 만들까요?"* → 스킬 생성
2. **(공통)** 팀 공통 맥락·이슈를 심고 → *"요즘 팀은 이걸 고민해요 / 이번 주 리포트 항목은 이거예요"* 먼저 챙김 → 내 질문·니즈와 연결해 방향 제시

**원칙:** 기록은 기본 **내 로컬에만**, 공유는 필요할 때 골라서(opt-in). 가장 작은 조각부터.
**최종 형태:** 사내 깃헙의 **우리팀 전용 플러그인**(Claude Code의 오마이클로드·비킷 같은)으로.

### ⚙️ 구현 — 실제로 만든 것

- **`detect_repeats.py` (반복 감지 엔진)** — 내 AI 활동 로그(`~/.claude/projects/**/*.jsonl`)를 읽어, **여러 날에 걸쳐 반복하는 요청**을 찾아낸다. 결과는 **로컬 전용**(`~/.giljabi/`)으로만 저장, 어디에도 전송하지 않는다.
- **실제 내 로그 2,077개로 검증** → 반복 업무가 실제로 자동 감지됨 (예: 매일 수동 실행하던 브리핑, 반복적으로 요청하던 데이터 정리, 반복 서버 실행 등 — *회사 세부 내용은 제외*).
- **`SKILL.md` (길잡이 인터페이스)** — 감지 결과를 대화체 제안으로 바꿔줌: *"‘○○ 실행’을 15일 반복하셨네요 — 자동실행으로 바꿀까요?"* → 수락하면 스킬/스크립트로 생성.
- **② 공통 맥락·이슈 브리핑 (구조 설계 + 데모 검증)** — 팀 채널에서 반복 일정·이슈성 메시지를 수집해 "요즘 팀 공통으로 챙길 것" 목록을 만들고, 내 질문과 **연관 맥락**을 연결해 방향을 짚어준다.

### 🧗 과정에서의 삽질

- **기존 자산 재사용의 함정:** 이미 있던 활동수집 스킬 출력을 쓰려 했는데, 그건 **개수·통계만** 저장하고 프롬프트 내용이 없었다 → 반복 "요청" 감지 불가. **원본 로그(jsonl)를 직접 읽는 방식**으로 전환. ("수집 스킬=통계용, 길잡이=내용분석용"으로 역할이 갈렸다.)
- **잡음과의 싸움:** 첫 감지 결과가 `[Request interrupted by user]` 같은 시스템 마커와 `그리고/지금/있어` 같은 연결어로 오염됐다 → ① 시스템 마커 필터 ② 한국어 불용어 대폭 보강 ③ 예시가 겹치는 중복 후보 합치기 로 해결하니 진짜 반복 업무가 드러났다.
- **보안 분리:** 실제 데이터가 닿는 부분이 생겼다 → 처음부터 **회사용(사내 비공개) / 발표용(제네릭)** 두 트랙으로 분리. 이 제출물엔 회사 내용 0%.

### ✅ 결과물

- 작동하는 **반복 감지 엔진** + **길잡이 스킬** (로컬 2,077개 로그에서 반복 업무 자동 감지 성공)
- 북극성 설계 문서 + 공통 맥락·이슈 브리핑 구조
- (스크린샷: `이미지첨부/` 폴더에)

### 💡 알게 된 인사이트 & 공유하고 싶은 내용

- 진짜 병목은 "AI 실력"이 아니라 **각자의 AI에 팀 공통 맥락이 없는 것**이었다. 그래서 프롬프트를 나눠주는 대신 **맥락을 자동으로 심는 그릇**을 만들어야 한다.
- 진짜 레버리지는 내 개인 자동화보다 **팀 공통 영역**에 있다 — 여럿이 같은 걸 반복하니까.
- 큰 시스템도 **가장 작은 조각(반복감지 한 루프)** 부터 만드니 손에 잡혔다.
