# 2주차 — 내 OS 구현하기 🚀

## 🎯 미션 1. 내 OS 만들기

**✅ 선택:** 내 삶을 돕는 OS (일·업무 OS)

> 1주차 "아침 브레인덤프 OS(머릿속 정리)"에서 한 걸음 더 나아가,
> **내 반복을 감지하고 팀 공통 이슈까지 먼저 알려주는** OS로 확장했다. 이름은 **길잡이**.

### 📐 기획 — 무엇을, 왜, 어떻게

**대상은 "AI를 잘 못 쓰는 사람".** MCP도 스킬도 모르고, 그냥 클로드랑 대화할 줄만 알고, 회사가 준 계정으로 클로드 코드에 들어갈 줄은 아는 사람.

그런 사람도 **팀 범용 스킬 하나**만 있으면:
- AI 활용법을 헤매며 **불필요한 질문을 하지 않고**,
- "우리 팀은 요즘 이런 걸 공통으로 고민한다"를 **먼저 알아채준다.**

설정할 건 딱 하나 — **활동수집.** 그것만 켜지면 무작정 대화하는 것과 **전혀 다르게** 움직인다.

**두 축:**
1. (개인) 내 반복 요청을 감지해 → "이거 매주 반복하시네요, 스킬로 만들까요?" 제안 → 스킬 생성
2. (공통) 팀이 공통으로 겪는 이슈를 수집해 → "요즘 팀은 이걸 고민해요" 먼저 브리핑 → 내 질문·보고서 니즈와 연결해 방향 제시

**원칙:** 기록은 기본 **내 로컬에만**. 필요할 때만 골라서 팀에 공유(opt-in). 가장 작은 조각부터.

### ⚙️ 구현 — 실제로 만든 것

- **`detect_repeats.py` (반복 감지 엔진)** — 내 AI 활동 로그(`~/.claude/projects/**/*.jsonl`)를 읽어, **여러 날에 걸쳐 반복하는 요청**을 찾아낸다. 결과는 **로컬 전용**(`~/.giljabi/`)으로만 저장, 어디에도 전송하지 않는다.
- **실제 내 로그 2,077개로 검증** → 반복 업무가 실제로 자동 감지됨 (예: 매일 수동 실행하던 브리핑 스킬, 반복적으로 요청하던 데이터 정리, 반복 서버 실행 등 — *회사 세부 내용은 제외*).
- **`SKILL.md` (길잡이 인터페이스)** — 감지 결과를 대화체 제안으로 바꿔줌: *"‘○○ 실행’을 15일 반복하셨네요 — 자동실행으로 바꿀까요?"* → 수락하면 스킬/스크립트로 생성.
- **② 공통 이슈 브리핑 (구조 설계 + 데모 검증)** — 팀 채널에서 장애·배포·이슈성 메시지를 수집해 "요즘 팀 공통 이슈" 목록을 만들고, 내 질문("이번달 문의 왜 늘었지?")과 **실제 연관 이슈**를 연결해 방향을 짚어준다.

### 🧗 과정에서의 삽질

- **기존 자산 재사용의 함정:** 이미 있던 활동수집 스킬(weekly-collect) 출력을 쓰려 했는데, 그건 **개수·통계만** 저장하고 프롬프트 내용이 없었다 → 반복 "요청" 감지 불가. **원본 로그(jsonl)를 직접 읽는 방식**으로 전환. ("수집 스킬=통계용, 길잡이=내용분석용"으로 역할이 갈렸다.)
- **잡음과의 싸움:** 첫 감지 결과가 `[Request interrupted by user]` 같은 시스템 마커와 `그리고/지금/있어` 같은 연결어로 오염됐다 → ① 시스템 마커 필터 ② 한국어 불용어 대폭 보강 ③ 예시가 겹치는 중복 후보 합치기 로 해결하니 진짜 반복 업무가 드러났다.
- **보안 분리:** 공통 이슈 데모에 회사 실데이터가 닿았다 → 처음부터 **회사용(사내 비공개) / 발표용(제네릭)** 두 트랙으로 분리. 이 제출물엔 회사 내용 0%.

### ✅ 결과물

- 작동하는 **반복 감지 엔진** + **길잡이 스킬** (로컬 2,077개 로그에서 반복 업무 자동 감지 성공)
- 북극성 설계 문서 + 공통 이슈 브리핑 구조
- (스크린샷: `이미지첨부/` 폴더에)

### 💡 알게 된 인사이트 & 공유하고 싶은 내용

- **"AI 잘하는 사람용 도구"가 아니라 "AI 못하는 사람의 바닥을 올려주는 도구"** 가 진짜 가치다. 무엇을 물어야 할지 모르는 사람에게, 묻기 전에 팀·내 맥락을 먼저 펼쳐주는 것.
- 만든 걸 자랑하는 게 아니라 **"어떻게 공동자산화하느냐"** 가 포인트. 각자 자기 것으로 키우게 하는 그릇을 만드는 것.
- 큰 시스템도 **가장 작은 조각(반복감지 한 루프)** 부터 만드니 손에 잡혔다.
