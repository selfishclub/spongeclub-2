# 1주차 — 나만의 OS 만들기 🛠️

> 미션을 진행하며 **과정과 결과물**을 기록해주세요. (다 못 채워도 OK, 한 것 위주로!)

## 🎯 미션 1. 내 OS 재료 찾기
> 인터뷰 스킬(아이데이션)로 "내 삶에 필요한 게 뭔지" 찾기

- **과정 (어떻게 찾았나):**
  인터뷰 스킬의 5스텝을 따라가며 하루를 되짚어봤다. "시간을 제일 많이 쓴 일 → 짜증 포인트 → 일 밖의 나 → 무게 재기 → OS가 된다면" 순으로 한 질문씩 답하다 보니, 처음엔 "할 일이 머릿속에만 있고 미뤄진다", "아이디어가 흘러간다" 정도로 흩어져 있던 게 점점 **한 가지**로 모였다. 일·생활·과제·살 것까지 영역은 달라도 전부 *"머릿속에 떠오른 게 어디에도 안 담겨 흘러간다"*는 같은 뿌리였다.

- **결과 — 🟡 내 OS 재료 카드:**

  | 항목 | 내용 |
  |---|---|
  | 영역 | 일 + 삶 둘 다 (머릿속 전체를 받아주는 한 곳) |
  | 걸리는 지점 | 떠오른 할 일·아이디어·살 것·일정이 흘러감. 카톡 '나에게 보내기'는 극히 일부고 다시 안 봄 |
  | 지금은 | 기억에 의존 |
  | OS가 된다면 | 아침 9시에 일·생활 전부 자연어로 나열 → ① 분류 ② 관리 ③ 알림 ④ 제안 ⑤ 흘러간 아이디어 디벨롭 ⑥ 유사 아젠다 연결 |
  | 한 문장 | "보내고 잊어도, 필요할 때 알아서 돌아온다" |
  | 첫 한 걸음 | 아침에 자연어로 쏟아낸 걸 분류해서 돌려주는 한 칸 |

- **느낀 점:**
  "딱히 불편한 게 없는데"가 아니라, **이미 던지고는 있었다**는 걸 깨달았다. 카톡 '나에게 보내기'에 보내놓고 다시 안 보는 그 무덤이 문제였다. 모으는 건 이미 하고 있으니, 진짜 필요한 건 **"때 되면 다시 꺼내주는 것"**이었다.

## 🧩 미션 2. 내 OS 기획
> 인터뷰 결과 + 세션 내용(흐민·배짱·키노) 활용해 기획

- **기획 내용 — "아침 브레인덤프 OS" 3단계 로드맵:**

  매일 아침 9시, 일이든 생활이든 떠오르는 걸 **자연어로 한 번에 쏟아내면**, AI가 받아서 정리·관리·제안까지 해주는 동반자형 OS. 한 번에 다 만들지 않고 3단계로 키운다.

  - **🥇 1단계 (이번 주):** 아침에 할 일을 자연어로 쭉 적으면 → `일 / 생활 / 아이디어 / 살 것 / 일정` 다섯 칸으로 **분류해 표로 돌려주기.** 알림·제안은 아직 없음. "쏟아내면 정리돼서 돌아온다"는 경험 하나만 완성.
  - **🥈 2단계:** 타이밍 있는 것(과제 마감·살 것)에 **알림**을 붙이고, 흘러간 아이디어를 다시 꺼내 **디벨롭**해주기.
  - **🥉 3단계:** 매일 쌓인 걸 보고 **먼저 제안**하고, **유사 아젠다끼리 연결**해주는 능동적 동반자 = 진짜 "내 OS".

- **막혔던 점 / 어떻게 풀었나:**
  처음엔 "분류·관리·알림·제안·디벨롭·연결"을 다 한 번에 만들고 싶어 그림이 너무 커졌다. 3단계로 쪼개고 **이번 주는 1단계(쏟아내기 → 분류)만** 끝내기로 범위를 좁히니 손에 잡혔다. 새 습관도 필요 없이, 이미 하던 '나에게 보내기'를 대체하는 것부터 시작한다.

## ⚙️ 미션 3. 내 OS 구현
> 실제로 만들어본 것 (클로드코드 '채널' 기능 활용 OK)

- **결과물:** ✅ 클로드 코드로 **'아침-브레인덤프' 스킬 제작 완료** (`.claude/skills/아침-브레인덤프/`). 두서없는 자연어 덩어리를 던지면 되묻지 않고 바로 `🗂 일 / 🏠 생활 / 💡 아이디어 / 🛒 살 것 / 📅 일정` 다섯 칸으로 분류해 표로 돌려준다. (1단계 = 분류까지. 알림·제안은 다음 단계.)
- **작동 방식:** "우유 사야하고, 과제 마감 금요일, 자동화 아이디어…" → 되묻지 않고 즉시 5칸 분류. 빈 칸은 생략, 내가 쓴 표현 그대로 유지, 날짜는 `(금)`처럼 옆에 표기.
