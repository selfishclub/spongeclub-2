# 1주차 — 나만의 OS 만들기 🛠️

> 미션을 진행하며 **과정과 결과물**을 기록해주세요. (다 못 채워도 OK, 한 것 위주로!)

## 🎯 미션 1. 내 OS 재료 찾기
> 인터뷰 스킬(아이데이션)로 "내 삶에 필요한 게 뭔지" 찾기
- **과정 (어떻게 찾았나):** `sponge-os-interview` 인터뷰 스킬 5스텝을 그대로 따라갔다. 처음엔 "딱히 만들 게 없는데" 싶었는데, 하루를 돌아보다 매달 **월급날 가계부 정산**에서 진짜 마찰점을 발견했다. 일이 아니라 '삶(돈 관리)' 쪽에서 나왔다.
- **결과:** 🟡 내 OS 재료 카드
  - **영역**: 삶 (돈 관리 · 월급날 가계부 정산)
  - **걸리는 지점**: 자동이체가 두 갈래(카드로 구독한 것 / 통장에서 바로 빠지는 것 — 주택청약·핸드폰요금 등)로 흩어져 있어, 매달 카드출금통장에 얼마를 미리 옮길지 다시 계산해야 함
  - **지금은**: 카드총액 − 생활비 = 초과분(쉬움) → 카드구독 자동이체분은 따로 추적해 미리 이체 → 새 구독 생기면 또 계산 → 합쳐서 송금
  - **한 문장**: "월급날 통장 정리하는 시간이 사라진다"
- **느낀 점:** 없는 줄 알았는데, 막상 하루를 들여다보니 매달 손 가던 게 딱 잡혀서 신기했다. "반복 + 살짝 귀찮음"이 OS 1순위 재료라는 말이 와닿았다.

## 🧩 미션 2. 내 OS 기획
> 인터뷰 결과 + 세션 내용(흐민·배짱·키노) 활용해 기획
- **기획 내용:** 재료 카드를 3단계로 쪼갰다.
  - **1단계 (이번 주)**: 자동이체 *목록*을 한 곳에. 항목별 `[이름 · 금액 · 분류(카드구독/통장직접) · 결제일]`을 넣으면 **카드구독분 합계**(= 월급날 카드출금통장에 미리 옮길 돈)를 자동 계산해주는 '자동이체 장부'.
  - **2단계**: 매달 "이번 달 옮길 돈 = 생활비 초과분 + 카드구독 자동이체분"을 한 번에 계산.
  - **3단계**: 새 구독 추가 시 "N원 더 옮겨야 함" 알림 (완전 자동화).
  - → ①이 토대, ②③이 그 위에 얹히는 구조라 **첫 한 걸음은 ①(목록) 부터**.
- **막혔던 점 / 어떻게 풀었나:** "OS를 거창하게 만들어야 하나" 막막했는데, 인터뷰가 '전체'가 아니라 '이번 주 1단계'로 범위를 좁혀줘서 시작점이 명확해졌다.

## ⚙️ 미션 3. 내 OS 구현
> 실제로 만들어본 것 (클로드코드 '채널' 기능 활용 OK)
- **결과물:** 🪙 **1초 가계부** — 월급날 통장 정리를 1초에 끝내는 계산기. 월급 2곳을 넣으면 ① 우리은행 직접 인출 ② 케이뱅크로 옮길 구독·자동이체분 ③ 투자 ④ 생활비를 빼고 → **남는 돈(토스뱅크 투자잉여)** 까지 자동으로 계산해준다. 항목·금액·결제일은 직접 추가/수정 가능하고, 데이터는 내 브라우저(localStorage)에만 저장된다.
- **만든 방법:** 클로드 코드로 HTML 한 장(`index.html`)으로 구현. 탭(요약/우리/케이/투자) 구조 + 자동 합계.
- **결과물 파일:** 같은 폴더 `index.html` (다운받아 브라우저로 열면 바로 작동)
