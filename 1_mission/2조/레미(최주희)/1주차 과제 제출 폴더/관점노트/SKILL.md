# 관점노트 (Perspective Note)

> "아침엔 시장을 떠먹고, 낮엔 공부를 쌓아, 흔들리지 않는 내 관점을 만든다."

주식·경제 유튜브를 소비만 하고 흘려보내지 않도록, 링크를 던지면 **요약 → 분류 → 누적**해
경제를 보는 관점을 쌓아주는 OS. (☀️ 아침 브리핑과 관심 섹터를 공유하는 '주식 인텔리전스 OS'의 축적 흐름)

## 관심 섹터 (씨앗 — 분류 기준, 새 섹터는 자동 추가)

- **AI반도체**: 엔비디아, SK하이닉스, 삼성전자, HBM 등
- **AI부품**: 기판·소재·부품·전력 등 AI 인프라 후방
- **피지컬 AI**: 현대차, 로보틱스·휴머노이드·자율주행
- **우주**: 스페이스X, KAI, 위성·발사체
- **방산**: 한화에어로스페이스, 현대로템, 수출 수주
- **시장환경(매크로)**: 지수(코스피·나스닥 등)·금리·유가·환율 등 시장 전반

## 저장 폴더 구조

`관점노트/` 아래에 저장한다. (이 SKILL.md 와 같은 위치 기준)

```
관점노트/
├── 0_인박스/           ← 새 요약이 일단 여기 (분류 애매하면 여기)
├── 영상노트/           ← 영상별 요약 노트 (원본 재료)
├── 종목섹터/           ← 섹터별 모아보기 (자동 생성·누적)
└── 패널/               ← 출연자별 추적 (자동 생성·누적)
```

## 링크를 받으면 (실행 순서)

1. **영상 내용 확보 (자막 우선)**
   - **yt-dlp로 자막을 뽑는다** (가장 정확). 임시 폴더에 받아서 저장소는 더럽히지 않는다:
     ```
     yt-dlp --write-auto-subs --write-subs --sub-langs "ko,ko-orig,en" \
            --skip-download --sub-format "vtt/best" -o "<임시폴더>/sub.%(ext)s" "<URL>"
     ```
     - yt-dlp 경로: winget 설치 시 `C:\Users\<사용자>\AppData\Local\Microsoft\WinGet\Links\yt-dlp.exe` (셸에 없으면 이 경로로 직접 호출)
     - 받은 `.vtt`는 타임스탬프·태그·중복이 많으므로, `<...>` 태그와 `-->` 줄 제거 + 연속 중복 제거해 **plain text**로 정리한 뒤 요약한다.
   - **폴백**: 자막이 없거나 실패(429 등)하면, WebFetch(제목·설명) + WebSearch로 최대한 파악하고 노트에 "자막 미확보 — 근거 제한적"이라 표시한다.

2. **핵심 추출**
   - 제목 / 패널(채널·출연자) / 날짜 / URL
   - 종목·섹터 태그 (위 씨앗 기준, 새로 나오면 태그 추가)
   - 핵심 요약 3~5줄
   - 주요 주장: 강세/약세 + 근거
   - 시장 환경 언급: 지수·금리·유가·매크로

3. **영상노트 저장** — `영상노트/{YYYY-MM-DD}_{제목요약}_({패널}).md` 로 아래 템플릿 저장:

```
# {영상 제목}
- 패널(출연자): {}
- 날짜: {}
- URL: {}
- 종목·섹터 태그: #{} #{} ...

## 핵심 요약 (3~5줄)
- ...

## 주요 주장
- 강세/약세: {}
- 근거: {}

## 시장 환경 언급 (지수·금리·유가·매크로)
- ...

## 내 메모 (선택)
-
```

4. **섹터·패널 파일에 누적** — 영상에서 나온 각 섹터마다:
   - `종목섹터/{섹터명}.md` 가 없으면 **새로 만들고**, 있으면 아래 한 줄을 **맨 위에 추가**:
     `- {날짜} · [{영상 제목}](../영상노트/파일명.md) · {그 영상이 이 섹터에 대해 한 말 한 줄} (강세/약세)`
   - `패널/{패널명}.md` 도 같은 방식으로 누적 (그 패널이 이번에 한 핵심 주장 한 줄).

5. **회신** — 무엇을 어디에 저장했는지 한 줄로 알려준다:
   `✅ 저장 완료 — 영상노트 1개 + [AI반도체, 시장환경]에 누적했어요.`

## 규칙

- **분류가 애매하면** `0_인박스/`에 넣고 "분류 확인 필요"로 표시. 억지로 섹터에 밀어넣지 않는다.
- **지어내지 않는다.** 자막/내용에서 확인된 것만 요약. 확인 안 되면 "확인 필요".
- 섹터 파일은 **누적**이 핵심 — 덮어쓰지 말고 최신이 위로 쌓이게.
- 이건 투자 판단 도구가 아니라 **관점 정리** 도구다. 매수/매도 추천 X.

## 다음 단계 (아직 미구현)

- **2단계**: 텔레그램 봇 연동 — 폰에서 링크 던지면 자동 저장 (강의의 채널 기능)
- **2단계**: 섹터·패널 파일 자동 요약해 "관점 추이" 뷰 만들기
- **나중**: 쌓인 관점으로 '내 관점 노트'(비공개) 정리 + 아침 브리핑과 상호 참조
