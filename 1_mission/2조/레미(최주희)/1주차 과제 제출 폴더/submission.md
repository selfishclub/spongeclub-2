# 1주차 — 나만의 OS 만들기 🛠️

> 미션을 진행하며 **과정과 결과물**을 기록해주세요. (다 못 채워도 OK, 한 것 위주로!)

## 🎯 미션 1. 내 OS 재료 찾기
> 인터뷰 스킬(아이데이션)로 "내 삶에 필요한 게 뭔지" 찾기

- **과정 (어떻게 찾았나):**
  인터뷰 스킬 5스텝을 따라가며 찾았다.
  - S1(오늘 하루): 배너 기획 / 고객사 메일 / 견적서 작성
  - S2(마찰점): 견적서 작성이 단순한데 손이 많이 감
  - S3(일 밖의 나): **주식 동향 보기** — 관심 섹터 흐름·미국 전날 장 보고 오늘 한국 장 예상하고 싶은데 업무 때문에 못 챙김. 유튜브 학습 영상도 저장만 하고 못 봄
  - S4~S5: 아침 브리핑 + 유튜브 요약 아카이빙을 묶은 '주식 인텔리전스 OS'

- **결과 (OS 재료 카드):**
  ```
  🟡 내 삶을 돕는 OS · 재료 카드
  · 영역      : 삶(관심사) — 업무에 밀려 놓치는 '주식 공부·시장 감' 챙기기
  · 걸리는 지점 : 관심 섹터 동향을 챙기고 싶은데 업무 때문에 못 봄.
                유튜브 학습 영상도 저장만 하고 끝까지 못 봄
  · 지금은    : 시간 날 때 증권 앱·뉴스 뒤져 직접 읽고 머릿속으로 예상
  · OS가 된다면 : ① 매일 아침 시장 브리핑 ② 유튜브 자동 요약·아카이빙
  · 한 문장   : "아침엔 시장을 떠먹고, 낮엔 공부를 쌓아, 흔들리지 않는 내 관점을 만든다"
  ```

- **느낀 점:**
  "딱히 만들 게 없는데" 싶었는데, 일에서 벗어나 삶을 보니 진짜 원하던 게 나왔다.
  불편함뿐 아니라 **'하고 싶은데 시간이 없어 놓치는 것'**도 훌륭한 OS 재료였다.

## 🧩 미션 2. 내 OS 기획
> 인터뷰 결과 + 세션 내용(흐민 세션·배짱 온맘) 활용해 기획

- **기획 내용:**
  흐민 세션의 5단계 루프(인풋→저장→연결→제안→발행)에서 영감을 받아,
  **하나의 OS, 두 흐름** 구조로 기획했다. → [설계 문서](관점노트-OS-설계.md)

  ```
          🗂️ 공통 뼈대: 관심 섹터 (AI반도체·에너지자원·우주·방산·시장환경 …)
  ☀️ 아침 브리핑 (소비)                🌙 관점노트 (축적)
   미국 전날 장+섹터 → 오늘 장 예상     유튜브 링크 → 요약 → 섹터·패널·날짜별 누적
            └──────── 같은 섹터 공유, 서로 먹여주는 선순환 ────────┘
  ```

  - 목적: 정보를 **분절분절** 받아 **줏대 없이 흔들리던 것**을, 전문가 인사이트를
    요약·연결해 **경제 보는 내 관점**으로 키우기.
  - 단계: 1단계(이번 주) 두 엔진 작동 → 2단계 텔레그램 연동·섹터 자동 연결 → 나중 '내 관점 노트'

- **막혔던 점 / 어떻게 풀었나:**
  ① 처음엔 '아침 브리핑(소비)'만 생각했는데, 흐민 세션을 보고 **'지식 축적'**이 내 진짜
  목적(줏대 세우기)에 더 맞다는 걸 발견 → 두 흐름을 **하나의 OS**로 합쳐 해결.
  ② 유튜브 내용을 클로드가 직접 못 읽는 문제 → 노트북LM은 공개 API가 없어서,
  **yt-dlp로 자막을 추출**해 요약하는 방식으로 해결.

## ⚙️ 미션 3. 내 OS 구현
> 실제로 만들어본 것 (클로드코드 '채널' 기능 활용 OK)

두 개의 클로드 코드 스킬을 만들고 **둘 다 실제로 돌려서 결과를 확인**했다.

- **① ☀️ 주식-브리핑 스킬** — "아침 브리핑 해줘" 하면 미국 전날 장+관심 섹터 뉴스를
  모아 "오늘 한국 장 예상" 한 장으로 정리 (WebSearch 기반).
  - [주식-브리핑/SKILL.md](주식-브리핑/SKILL.md) · 실행 샘플: [샘플-브리핑-2026-07-02.md](주식-브리핑/샘플-브리핑-2026-07-02.md)

- **② 🌙 관점노트 스킬** — 유튜브 링크를 던지면 **yt-dlp로 자막 추출 → 요약 →
  종목·섹터/패널/날짜별 MD로 자동 분류·누적**. 새 섹터는 자동 생성.
  - [관점노트/SKILL.md](관점노트/SKILL.md)
  - **실제 테스트**: 오건영 단장 영상(지식인초대석 EP.143) 자막 1.8만 자를 요약 →
    [영상노트](관점노트/영상노트/2026-07-02_반도체말고-조용히뜨는자산_오건영.md) +
    섹터파일([시장환경](관점노트/종목섹터/시장환경-매크로.md)·[에너지자원](관점노트/종목섹터/에너지-자원.md)·[AI반도체](관점노트/종목섹터/AI반도체.md)) +
    [패널/오건영](관점노트/패널/오건영.md) 자동 생성·누적 확인.
  - **배운 점**: 제목만 봤을 땐 "뜨는 자산=금"이라 추측했는데, 자막 전문을 보니
    실제는 **에너지·자원**이었다. 자막 기반 요약의 중요성을 직접 확인.

- **결과물 요약:** 아침엔 브리핑으로 시장을 떠먹고, 공부한 영상은 관점노트에 쌓여
  '경제 보는 눈' 재료가 된다. 같은 관심 섹터를 공유하는 하나의 OS.
