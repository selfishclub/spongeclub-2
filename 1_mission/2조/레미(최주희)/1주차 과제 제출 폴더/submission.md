# 1주차 — 나만의 OS 만들기 🛠️

> 미션을 진행하며 **과정과 결과물**을 기록해주세요. (다 못 채워도 OK, 한 것 위주로!)

## 🎯 미션 1. 내 OS 재료 찾기
> 인터뷰 스킬(아이데이션)로 "내 삶에 필요한 게 뭔지" 찾기

- **과정 (어떻게 찾았나):**
  인터뷰 스킬로 5스텝을 따라가며 찾았다.
  - S1(오늘 하루): 시간을 많이 쓴 일 — 프로모션 페이지 핵심 혜택 뽑아 배너 기획 / 고객사 메일 회신 / 견적서 작성
  - S2(마찰점): 견적서 작성이 단순한데 손이 많이 감
  - S3(일 밖의 나): **주식 동향 보기** — 관심 종목·섹터 흐름, 미국 전날 장 보고 오늘 한국 장 예상하는 걸 하고 싶은데 업무 때문에 자꾸 못 챙김. 유튜브 학습 영상도 저장만 하고 못 봄
  - S4(무게 재기): 견적서(일) vs 주식 동향(삶) 중 → **주식 동향** 선택
  - S5(딱 하나): 아침 장 브리핑 + 유튜브 요약 아카이빙을 묶은 '주식 인텔리전스 OS'

- **결과 (OS 재료 카드):**
  ```
  🟡 내 삶을 돕는 OS · 재료 카드
  · 영역      : 삶(관심사) — 업무에 밀려 놓치는 '주식 공부·시장 감' 챙기기
  · 걸리는 지점 : 관심 종목·섹터 동향을 챙기고 싶은데 업무 때문에 못 봄.
                유튜브 학습 영상도 저장만 하고 끝까지 못 봄
  · 지금은    : 시간 날 때 증권 앱·뉴스 여기저기 열어서 미국 전날 장·관심 종목
                직접 찾아 읽고, 머릿속으로 오늘 한국 장 예상
  · OS가 된다면 : ① 매일 아침 '미국 전날 장 + 관심 섹터 → 오늘 한국 장 예상' 브리핑
                ② 못 본 유튜브 영상 자동 요약 → 패널·일자별 아카이빙
  · 한 문장   : "출근길 5분이면 오늘 장이 손에 잡힌다"
  · 첫 한 걸음 : 클로드 코드로 '주식-브리핑' 채널부터 (아침 브리핑 A안)
  ```

- **느낀 점:**
  "딱히 만들 게 없는데" 싶었는데, 일 얘기에서 벗어나 삶 쪽을 들여다보니 진짜 원하던 게 튀어나왔다. 불편함이 아니라 **'하고 싶은데 시간이 없어 놓치는 것'**도 훌륭한 OS 재료라는 걸 알게 됐다.

## 🧩 미션 2. 내 OS 기획
> 인터뷰 결과 + 세션 내용 활용해 기획

- **기획 내용:**
  카드를 3단계로 쪼개고, 이번 주는 1단계만 만들기로 함.
  - **1단계 (이번 주)**: '아침 장 브리핑' — 미국 전날 장 마감 + 관심 섹터(반도체·피지컬AI·우주/방산) 뉴스를 모아 "오늘 한국 장 예상" 한 장으로. **텔레그램 연동 없이** 브리핑이 제대로 나오는지부터 확인.
  - **2단계**: 이 브리핑을 텔레그램 봇으로 매일 아침 자동 전송
  - **3단계**: 유튜브 학습 영상 자동 요약 + 패널·일자별 아카이빙 연동 (영상에서 언급된 종목이 아침 브리핑에도 반영되는 완성형)
  - 관심 섹터: 반도체(SK하이닉스·삼성전자·엔비디아) / 피지컬AI(현대차) / 우주·방산(KAI·한화에어로·스페이스X)

- **막혔던 점 / 어떻게 풀었나:**
  A(아침 브리핑)와 B(유튜브 아카이빙) 둘 다 만들고 싶어 욕심이 났는데, 한꺼번에 하면 무거워짐. **1단계를 A 하나로 좁히고** 나머지는 2·3단계로 미뤄 이번 주에 끝낼 수 있는 크기로 만들었다.
