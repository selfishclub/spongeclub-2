# 주식 인텔리전스 OS — 설계 문서

> 작성일: 2026-07-02 · 작성자: 레미(최주희), 2조
> 하나의 OS, 두 개의 흐름: ☀️ 아침 브리핑(소비) + 🌙 관점노트(축적)

## 왜 만드는가 (Context)

지금은 주식·경제 정보를 **분절분절** 받아들여서 **줏대 없이 흔들릴 때가 많다.**
전문가 콘텐츠(유튜브)를 통해 시장 흐름·섹터 동향·주가 추이를 **요약하고 연결해서,
경제를 보는 나만의 관점을 키우는 것**이 이 OS의 목적이다.

- 단순 영상 아카이브가 아니다. 흩어진 조각을 **연결해 관점을 만드는** 도구다.
- 흐민 세션의 5단계 루프(인풋→저장→연결→제안→발행)를 내 버전으로 가져왔다.

## 전체 그림 — 하나의 OS, 두 흐름

같은 **관심 섹터 목록**을 공유하는 두 흐름이 선순환한다.

```
        🗂️ 공통 뼈대: 관심 섹터
 (AI반도체·AI부품·피지컬AI·우주·방산·시장환경)

☀️ 아침 브리핑 (소비, 이미 만듦)     🌙 관점노트 (축적, 이번에 만들 것)
 미국 전날 장 + 섹터 뉴스            유튜브 링크 던지면 → 요약 →
 → 오늘 한국 장 예상 한 장           섹터·패널·날짜별 MD로 누적
          └──────── 서로 먹여줌 ────────┘
  · 브리핑의 관심 섹터 = 관점노트의 분류 섹터 (같은 목록)
  · 관점노트에 쌓인 내 관점이 나중에 브리핑에 반영됨
```

| | ☀️ 브리핑 (기존) | 🌙 관점노트 (신규) |
|---|---|---|
| 언제 | 아침마다 | 영상 볼 때 수시로 |
| 방향 | 정보가 나한테 옴 (소비) | 내가 정보를 넣음 (축적) |
| 결과 | "오늘 장 이렇겠다" | "나는 이 섹터를 이렇게 본다" |

## 한 문장

"아침엔 시장을 떠먹고, 낮엔 공부를 쌓아, 흔들리지 않는 내 관점을 만든다."

## 컨셉 — 흐민 루프의 내 버전

| 단계 | 내 OS | 이번 범위 |
|---|---|---|
| ① 인풋 | 텔레그램에 주식 유튜브 링크 던지기 | 2단계 (엔진 먼저) |
| ② 저장 | 영상 요약 → 종목·섹터·패널·날짜·시장환경 태그로 MD 저장 | **1단계 (이번 주)** |
| ③ 연결 | 섹터별·패널별로 조각을 모아 관점 추이 보기 | 2단계 |
| ④ 제안 | 내 관점 요약 리포트 제안 | 나중 |
| ⑤ 내 관점 노트 | 비공개 '내 생각' 정리 (흔들림의 해독제, 공개 발행 아님) | 나중 |

## 만드는 순서 (접근법 🅑 채택)

**요약 엔진 먼저 → 텔레그램은 그다음.**
1. "링크 붙여넣으면 → 요약 → 섹터별 MD 저장" 스킬을 만들어 클로드 코드에서 바로 테스트
2. 작동 확인 후, 텔레그램을 입력 통로로 연결 (강의의 채널 기능)

이유: 핵심 가치(요약·분류·누적)가 먼저 작동하면, 텔레그램 세팅에서 막혀도 좌절하지 않는다.

## 이번 주 1단계 — 만들 것

**입력**: 유튜브 링크를 클로드 코드에 붙여넣기 (텔레그램 없이)
**처리**: 영상 내용 요약 → 종목·섹터 / 패널 / 날짜 / 시장환경(지수·금리·유가) 추출
**출력**: MD 노트로 자동 저장 + 관련 섹터 파일에 누적

### 폴더 구조

```
관점노트/
├── 0_인박스/           ← 새 요약이 일단 여기
├── 영상노트/           ← 영상별 요약 (원본 재료)
│   └── 2026-07-02_엔비디아-실적-분석_(패널명).md
├── 종목섹터/           ← 섹터별 모아보기 (자동 생성·누적)
│   ├── AI반도체.md
│   ├── AI부품.md
│   ├── 피지컬AI.md
│   ├── 우주.md
│   ├── 방산.md
│   └── 시장환경-매크로.md
└── 패널/               ← 출연자별 추적 (자동 생성·누적)
    └── (패널명).md
```

- 섹터 MD 파일은 **미리 안 만든다.** 영상에 처음 등장하면 OS가 자동 생성하고, 다음부터 이어서 누적.
- 시작 씨앗 섹터: AI반도체 / AI부품 / 피지컬AI / 우주 / 방산 / 시장환경(매크로)

### 영상 노트 템플릿 (내용 요약하며 유연하게 보완)

```
# [영상 제목]
- 패널(출연자):
- 날짜:
- URL:
- 종목·섹터 태그: #AI반도체 #엔비디아 ...

## 핵심 요약 (3~5줄)

## 주요 주장
- 강세/약세:
- 근거:

## 시장 환경 언급 (지수·금리·유가·매크로)

## 내 메모 (선택)
```

## 범위 밖 (이번 주 안 함)

- 텔레그램 봇 연동 (2단계)
- 섹터·패널 자동 연결/추이 뷰 (2단계 — 태그만 잘 박아두면 나중에 쉬움)
- 내 관점 리포트 제안·정리 (나중)

## 성공 기준 (검증)

- 유튜브 링크 하나를 넣으면 → 템플릿대로 채운 영상노트 MD가 `영상노트/`에 생기고
- 언급된 섹터의 `종목섹터/OO.md`와 `패널/OO.md`에 요약 한 줄이 누적된다
- 새 섹터가 나오면 파일이 자동으로 새로 만들어진다
- 며칠 쌓은 뒤 `종목섹터/AI반도체.md`만 열어도 "그동안 반도체를 어떻게 봤나"가 한눈에 보인다
