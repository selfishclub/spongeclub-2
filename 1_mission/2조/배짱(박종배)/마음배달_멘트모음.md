# 마음배달 · 배짱 멘트 모음 (운영 스크립트)

2조 마니또 이벤트 「마음배달」에서 배짱이 뿌리는 멘트 세트.
흐름: **오프닝 → 배정 DM → 주간 미션 → 깜짝 미션 → 추측 스레드 → 추리 대회 접수 → 3주차 발표.**

> 표기: **[확정]** = 직접 작성한 최종본 · **[초안]** = 톤 맞춰 채운 초안(확정본 있으면 교체)
> 매체: Slack. 실제 올릴 땐 이모지를 `:shushing_face:` 같은 코드로 바꿔도 됨.

---

## ① [초안] 오프닝 공지 — 2주차 조별모임/채널

```
🧀💌 [마음배달] 오늘부터 우리, 서로의 다정한 우체부가 됩니다

이번 주 우리 2조엔 작은 비밀이 하나 생겨요.
각자 '몰래 챙길 한 사람(마니또)'이 정해지고,
티 안 나게 응원하고 챙기는 거예요. 🤫

거창할 것 없어요 — 이모지 하나, 다정한 댓글 한 줄, 셸 한 스푼이면 충분.
그리고 3주차 오프라인에서 "누가 내 우체부였게~?" 대공개 🎉

곧 각자에게 '봉투'가 도착해요. 열어보면 내 마니또가 누군지 알 수 있어요 💌
```

## ② [초안] 마니또 배정 안내 — 킥오프 DM (개인별)

```
🤫 [마음배달] OO님, 봉투가 도착했어요 💌

아래 링크를 열면 이번 주 'OO님이 몰래 챙길 마니또'와 미션이 담겨 있어요.
👉 (봉투 링크)  https://maeumbaedal.vercel.app

· 정체는 3주차 발표 전까지 살짝 비밀! 🤫
· 챙김은 개인 DM 말고 '채널 안에서' 가볍게 한 스푼씩
· 셸은 보내는 사람이 표시되니, 티 안 나게 살살 😏
```

## ③ [초안] 주간 앵커 미션 안내 — 채널

```
💌 [마음배달] 이번 주 다정 미션, 한 스푼씩만!

봉투 속 미션, 부담 갖지 마세요. 딱 이 정도면 완벽해요 👇
· 내 마니또가 조용한 하루면 → "잘 지내죠? 🤫" 한 줄
· 뭔가 해냈으면(아주 작은 것도!) → 제일 먼저 🎉 눌러주기

완벽 말고 꾸준. 오늘 한 스푼, 내일 한 스푼 🥄
```

## ④ [초안] 주중 깜짝 미션 — 툭 투입용 (골라 쓰기)

```
🎁 깜짝 미션! 오늘 안에, 내 마니또 글에 '진심 담긴 댓글 한 줄' 남기기 (티 안 나게 🤫)
🎁 깜짝 미션! 지금 채널에서 아무나 붙잡고 응원 이모지 3개 뿌리기 — 우체부 정체 흐리기 작전 😏
🎁 깜짝 미션! 내 마니또가 좋아할 것 같은 짤/링크 하나 '툭' 던지기
```

## ⑤ [확정] "누굴까?" 추측 스레드 오프너 — 주 중반

```
🤫 [마음배달] 중간 점검 — 누가 누구의 우체부일까?

벌써 며칠째, 우리는 서로의 다정한 우체부로 지내고 있어요.
이모지 하나, 다정한 댓글, 슬쩍 날아온 셸 하나…
채널을 잘 보면 '어? 저 둘 수상한데?' 싶은 조합, 보이지 않나요? 👀

여긴 가볍게 떠보는 잡담 스레드예요.
"○○님 오늘 유난히 △△님 챙기던데?" 같은 심증, 편하게 던져보세요.

👉 진짜 승부는 따로 있어요!
   전체 배정을 예측해서 맞히는 '마니또 추리 대회'가 곧 열려요.
   가장 많이 맞힌 분께는 배짱's 특별 선물이 🎁
   (제출 방법은 곧 공지할게요. 지금부터 채널 잘 봐두세요 👀)

⚠️ 우체부 여러분, 너무 쉽게 들키면 재미없어요.
   가끔 페이크도 섞으면서 살살 즐겨요 😏
```

## ⑥ [초안] 추리 대회 제출 공지 — (⑤에서 예고한 "곧 공지")

```
🏆 [마음배달] 마니또 추리 대회, 접수 시작!

전체 배정(누가 → 누구)을 예측해서 제출하세요.
가장 많이 맞힌 분께 배짱's 특별 선물 🎁

· 제출: (추리 웹앱 링크 / 또는 배짱에게 DM으로 봉인 제출)
· 마감: 3주차 발표 직전까지
· 규칙: 제출 후 수정 없기! (봉인이에요 🤫)
· 발표: 3주차 오프라인에서 정답 공개 + 최다 정답자 시상
```

## ⑦ [확정] 3주차 마니또 발표 진행문

```
🎉 [마음배달] 대망의 마니또 발표 — 진행 순서

━━━ 진행자 오프닝 ━━━
"지난 한 주, 우리 열두 명은 각자 누군가의 다정한 우체부였어요.
누가 나를 챙겼을지, 그리고 누가 누구의 우체부였을지…
이제 전부 밝힐 시간이에요 🤫"

━━━ 1라운드 · 정체 공개 (한 명씩) ━━━
1) "저는 __님의 우체부였습니다!" 하고 본인이 챙긴 마니또 공개
2) 마니또에게 한마디 — 어떤 마음으로 챙겼는지 (30초)
3) 준비한 선물(5천~1만 원) 직접 건네기 🎁
   → 공개될 때마다 진행자가 '정답 배정표'를 채워간다

━━━ 2라운드 · 추리 대회 채점 & 시상 🏆 ━━━
· 완성된 정답 배정표로 제출 예측 채점 (웹앱이면 정답 입력 → 자동 순위)
· 각자 몇 개 맞혔는지 확인
· 최다 정답자 → 배짱's 특별 선물 🎁
· 동점이면 즉석 스피드 타이브레이크 ("3번은 누구?" 먼저 맞히기)

━━━ 3라운드 · 소감 한 스푼 (자유) ━━━
· 내 우체부, 맞혔나요?
· 가장 뭉클했던 챙김 한 순간
· 받았을 때 제일 좋았던 셸/댓글

━━━ 마무리 ━━━
"익명이라 더 솔직했던 우리의 일주일.
누가 챙겼는지는 비밀이었지만, 챙기는 마음은 전부 진심이었죠.
다음엔 더 많은 사람과 이 마음, 배달해봐요 💌"
```
