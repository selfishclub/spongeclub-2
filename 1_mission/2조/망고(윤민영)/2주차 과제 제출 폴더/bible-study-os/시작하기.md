# Come Away — 설치 가이드

> 성경 본문 하나 입력하면 번역 비교 + 단어 분석 + 묵상 기록까지 한 화면에

---

## 필요한 것

- Mac (macOS)
- 인터넷 연결

---

## 설치 (딱 두 가지만)

### 1. Claude Code 설치

아래 링크에서 Mac용 앱 다운로드 후 로그인

**https://claude.ai/download**

> 설치 후 한 번 실행해서 로그인까지 완료해두세요.

---

### 2. 터미널에서 주소 등록 (한 번만)

Mac에서 터미널 앱을 열고 아래 명령어를 그대로 붙여넣기:

```bash
echo "127.0.0.1 psalm1331" | sudo tee -a /etc/hosts
```

비밀번호 물어보면 Mac 로그인 비밀번호 입력 (입력해도 화면에 안 보이는 게 정상)

---

## 실행

1. `묵상-os.py` 파일을 아무 폴더에 저장
2. 터미널에서 해당 폴더로 이동 후:

```bash
python3 묵상-os.py
```

3. 브라우저에서 접속:

**http://psalm1331:1331**

---

## 사용법

1. 오늘 묵상할 성경 본문 입력 (예: `베드로전서 3:15`, `Romans 8:28`)
2. **분석하기** 버튼 → 30초~1분 기다리면 번역 비교 + 단어 분석 자동으로 나옴
3. 아래 묵상 내용 + 액션 플랜 작성
4. **기록 저장** → **카카오톡에 공유하기**로 그룹채팅에 나눔
