# 1주차 — 나만의 OS 만들기 🛠️

> 미션을 진행하며 **과정과 결과물**을 기록해주세요. (다 못 채워도 OK, 한 것 위주로!)

## 🎯 미션 1. 내 OS 재료 찾기
> 인터뷰 스킬(아이데이션)로 "내 삶에 필요한 게 뭔지" 찾기
- **과정 (어떻게 찾았나):** Claude와 5단계 인터뷰를 통해 하루를 돌아봤다. 처음엔 "딱히 불편한 게 없는데"였는데, 아침 루틴을 하나씩 짚어가다 보니 성경 묵상 흐름 전체가 탭 이동과 복붙으로 분산돼 있다는 걸 발견했다.
- **결과:**

  | 항목 | 내용 |
  |------|------|
  | 영역 | 삶 — 아침 묵상 시간 |
  | 걸리는 지점 | 4개 번역본 읽기, 단어·원어 공부, 묵상 기록이 전부 흩어져 있음 |
  | 지금은 | bible.com 열기 → 한글 읽기 → 번역본 2개만 비교 → 모르는 표현 복붙해서 Claude에 질문 → 줄글 답변 스크롤 → 묵상·적용점 따로 기록 |
  | OS가 된다면 | 본문만 넣으면 4개 번역본 비교 + 단어 뜻 + 원어 정리가 표로 한 번에 나오고, 막힐 때 바로 질문, 묵상과 적용점이 날짜별로 쌓임 |
  | 한 문장 | "탭 열고 복붙하던 시간이 사라지고, 묵상 자체에 집중하는 아침이 된다 — 오늘의 묵상은 한 줄과 몇 개의 단어로 남는다" |
  | 첫 한 걸음 | 성경 본문을 입력하면 4개 번역본 비교표 + 주요 단어 원어·뜻을 표로 뽑아주는 Claude 채널 하나 만들기 |

- **느낀 점:** 불편함인 줄 몰랐던 게 불편함이었다. 매일 하는 일이라 그냥 당연하게 여겼는데, 인터뷰로 짚어보니 아침마다 꽤 많은 에너지를 탭 이동과 복붙에 쓰고 있었다.

## 🧩 미션 2. 내 OS 기획
> 인터뷰 결과 + 세션 내용(흐민·배짱·키노) 활용해 기획
- **기획 내용:**

  **이름:** 아침 묵상 OS (성경 묵상 채널)

  **핵심 기능 3가지:**

  1. **번역본 비교표** — 본문 입력 시 개역개정 / 새번역 / NKJV / NASB 4개를 나란히 표로 출력
  2. **단어·원어 학습표** — 주요 단어의 헬라어·히브리어 원어 뜻 + 영어 뉘앙스를 표로 정리
  3. **묵상 기록** — 날짜 / 본문 / 한 줄 묵상 / 기억할 단어 몇 개를 누적 저장

  **단계별 로드맵:**

  | 단계 | 내용 |
  |------|------|
  | 1단계 (이번 주) | Claude 채널에 본문 입력 → 4개 번역본 비교표 + 단어 원어·뜻 표 자동 출력 |
  | 2단계 | 묵상 한 줄 + 키워드를 날짜별로 기록하는 로그 추가 |
  | 3단계 | 전체 묵상 OS — 본문 입력부터 기록까지 한 화면에서 완결 |

- **막혔던 점 / 어떻게 풀었나:** VS Code, Claude, GitHub을 다 켜놓고도 정말 뭐부터 어떻게 해야 하는지 아무도 알려주지 않으니 답답하고 속상했다. 과제 시작 자체를 어떻게 해야 하는지 몰라서 한참 멍하니 있었는데, 결국 Claude에게 "뭘 해야 하는지 모르겠다"고 털어놓으면서 인터뷰를 시작했고, 대화하다 보니 방향이 잡혔다.
