## 🎯 미션 1. 내 OS 만들기
> **[ 내 삶을 돕는 OS ]** 또는 **[ 콘텐츠 OS ]** 중 하나를 선택해 완성해주세요.

**✅ 선택:** 내 삶을 돕는 OS

### 📐 기획
> 무엇을, 왜, 어떻게 만들지

- **무엇을**: 매일 정해진 시간마다 다음 할 일을 텔레그램으로 알려주는 자동 알림봇
- **왜**: 하루에 복부 운동 + 해부도 그리기, 블로그, 쓰레드, 유튜브, neonskill.com 등 여러 채널을 운영하다 보니 흐름이 끊기는 문제가 있었다. 머릿속으로 루틴을 알고 있어도 실제로 시간을 놓치는 경우가 많아 알림이 필요했다.
- **어떻게**: Claude Code 인터뷰로 니즈를 발견 → 텔레그램 봇 생성 → Python 스크립트 작성 → macOS cron으로 자동화

### ⚙️ 구현
> 실제로 만든 것

- 텔레그램 봇: https://t.me/@neontechbot
- Python 스크립트가 현재 시간을 확인해 해당 할 일 메시지를 전송
- macOS cron으로 매일 8·9·10·11·12·14시에 자동 실행

**알림 스케줄:**

| 시간 | 알림 내용 |
|------|---------|
| 08:00 | 복부 운동 + 해부도 그리기 + 카페 업로드 |
| 09:00 | 블로그 포스팅 |
| 10:00 | 쓰레드 글 올리기 |
| 11:00 | 첫끼 식사 |
| 12:00 | 유튜브 촬영 + 편집 |
| 14:00 | neonskill.com 글 업로드 |
| 매주 일요일 14:00 | 오르가즘 교육 강의 🎓 |

### 🧗 과정에서의 삽질
> 막혔던 지점, 시도한 방법, 어떻게 풀었는지 솔직하게

- URL에 대괄호 `[토큰]`을 그대로 넣어서 404 오류 → 대괄호 제거로 해결
- 소문자 `l`과 대문자 `I`가 폰트에서 똑같이 보여서 401 오류 → BotFather에서 토큰 다시 복사해서 해결
- 봇에게 `/start` 메시지를 먼저 보내야 Chat ID가 생성된다는 것을 몰랐음 → 순서를 지키니 해결

### ✅ 결과물
> 완성한 것 / 작동 화면

- 텔레그램 봇 링크: https://t.me/@neontechbot
- 매일 정해진 시간에 텔레그램 알림이 자동으로 발송되고 있음

### 💡 알게 된 인사이트 & 공유하고 싶은 내용
> 하면서 깨달은 것, 크루들과 나누고 싶은 것

- 인터뷰를 통해 내 루틴을 글로 정리하니 내가 얼마나 많은 일을 하고 있는지 객관적으로 보였다.
- "알림 하나"를 만들기 위해 봇 토큰, Chat ID, API, cron까지 여러 개념을 자연스럽게 배웠다.
- 코딩을 몰라도 Claude Code와 함께하면 실제로 작동하는 것을 만들 수 있다는 걸 직접 경험했다.
