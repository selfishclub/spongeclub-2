# 1주차 — 나만의 OS 만들기 🛠️

> 미션을 진행하며 **과정과 결과물**을 기록해주세요. (다 못 채워도 OK, 한 것 위주로!)

## 🎯 미션 1. 내 OS 재료 찾기
> 인터뷰 스킬(아이데이션)로 "내 삶에 필요한 게 뭔지" 찾기
- **과정 (어떻게 찾았나):** Claude Code와 인터뷰를 통해 나의 하루 루틴을 정리했다. 아침 7시 기상 후 복부 운동 + 해부도 그리기 → 블로그 포스팅 → 쓰레드 → 식사 → 유튜브 → neonskill.com 글쓰기 순서로 움직이고 있음을 확인했다.
- **결과:** 핵심 니즈 발견 — "매시간 할 일을 알려주는 알림 시스템이 필요하다." 카카오톡 + 텔레그램으로 받고 싶고, A형 말투(⏰ N시가 됐어요! ~할 시간이에요.) 선호.
- **느낀 점:** 머릿속으로는 루틴을 알고 있었지만 글로 정리하니 내가 얼마나 많은 일을 하고 있는지 보였다. 알림이 없으면 흐름이 끊기는 이유도 파악됐다.

## 🧩 미션 2. 내 OS 기획
> 인터뷰 결과 + 세션 내용 활용해 기획
- **기획 내용:** "매시간 나를 깨워주는 개인 스케줄러 봇" — 텔레그램 봇이 정해진 시간마다 다음 할 일을 알려준다. 별도 앱 없이 텔레그램만으로 작동. 일요일 14시엔 강의 알림 추가.
- **막혔던 점 / 어떻게 풀었나:** 토큰을 URL에 입력할 때 대괄호를 그대로 넣어서 404 오류가 났다. 대괄호 제거 후 해결. 이후 소문자 l / 대문자 I 혼동으로 401 오류가 났고, BotFather에서 토큰을 다시 복사해서 해결했다.

## ⚙️ 미션 3. 내 OS 구현
> 실제로 만들어본 것
- **결과물:** 텔레그램 자동 알림봇 완성
  - Python 스크립트(`sungdaeje_alarm.py`)가 시간을 확인해서 해당 할 일 메시지를 텔레그램으로 전송
  - macOS cron으로 매일 8시·9시·10시·11시·12시·14시에 자동 실행
  - 일요일 14시에는 교육 강의 알림 별도 발송
- **알림 스케줄:**

| 시간 | 알림 내용 |
|------|---------|
| 08:00 | 복부 운동 + 해부도 그리기 + 카페 업로드 |
| 09:00 | 블로그 포스팅 |
| 10:00 | 쓰레드 글 올리기 |
| 11:00 | 첫끼 식사 |
| 12:00 | 유튜브 촬영 + 편집 |
| 14:00 | neonskill.com 글 업로드 |
| 매주 일요일 14:00 | 오르가즘 교육 강의 🎓 |
