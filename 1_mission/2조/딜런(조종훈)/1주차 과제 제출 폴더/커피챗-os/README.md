# 커피챗 OS ☕

가고 싶은 곳이 생겼는데 "누구한테, 무슨 명분으로 연락하지?"에서 멈춰본 적 있다면.
이 OS가 **누구를 · 어떤 명분으로 · 어떤 채널로 · 뭐라고** 연락할지를 확신 있게 짚어준다.

## 누구에게?
이직·네트워킹·커피챗을 하고 싶은데 첫 연락이 막막한 사람 누구나.
자기 `profile.md`만 채우면 직무 무관하게 작동한다.

## 어떻게 쓰나
1. `profile.md`에 내 경력·강점·관심사·인맥 결을 한 번 채운다
2. OS 호출 → "누구/어느 회사에 커피챗하고 싶어" 입력
3. 결과 받기 ↓

## 무엇이 나오나
- 🔎 **상대를 입체적으로** — 5개 앵글(누구인지·요즘 꽂힌 것·회사 방향·나와의 접점·맞는 채널)로 수집
- 🎯 **누구를** — 커피챗 1~3순위 + 접근 난이도 + 지인 연결 히든카드
- 💡 **어떤 명분** — 상대가 "요즘 꽂힌 것"에서 자연스러운 연락 명분 생성 (이게 핵심)
- 📨 **어떤 채널 + 첫 메시지 초안** — 복붙해서 살짝 고치면 바로 보낼 수 있게

## 정보 적은 사람/회사도 OK
정보가 흩어져 있어도 5개 앵글로 여러 매체(링크드인·유튜브 인터뷰·뉴스·X)를 긁는다.
> 기본은 웹검색이라 별도 도구 없이 작동. 리서치 도구(Apify 등)가 있으면 더 깊게, 없어도 같은 구조.

## 왜 만들었나
BD는 "남을 위한 명분"은 잘 만드는데, 정작 "나를 위한 커피챗"엔 명분이 없어서 멈춘다.
회사가 가져간 시간 안에서, 내 미래를 위한 만남을 주도적으로 만들기 위해 만들었다.

## 숨은 목적
누굴 만나고 무슨 얘기에 끌렸는지가 `log.md`에 쌓이면 — 내가 어디로 가고 싶은지가 데이터로 드러난다.
커피챗 도구이자, 방향 탐색 도구.
