# 커리어 OS — 회사 타겟 모드 🎯

가고 싶은 회사가 생겼을 때, **리서치 → 핏 점수 → 액션(커피챗 or 지원)**까지 한 흐름으로.

## 누구에게?
이직·커리어를 주도적으로 준비하고 싶은 사람 누구나. 자기 `profile.md`만 채우면 직무 무관하게 작동.

## 어떻게 쓰나
1. `profile.md`에 경력·강점·관심사를 채운다 (이력서·포폴 붙여주면 OS가 정리해줌)
2. OS 호출 → "○○ 회사 가고 싶어" 입력
3. 결과 ↓

## 무엇이 나오나
- 🔎 **회사 리서치 (5앵글)** — 회사·타이밍·핵심인물·링크드인 사람·접근채널
- 📊 **JD 핏 점수** — 내 프로필 vs JD 점수화 (근거 포함)
- 🧭 **유사 기업 + 산업 트렌드** — 다른 선택지 + 방향 판단 재료
- 🚀 **액션 분기**
  - 커피챗 → 누구를·명분·채널·첫 메시지
  - 지원 → 보완 경험 + 1~2개월 플랜

## 정보 적은 회사도 OK
5앵글로 여러 매체(기사·유튜브·링크드인)를 긁는다.
> 기본은 웹검색이라 도구 없이 작동. 리서치 도구(Apify 등)가 있으면 더 깊게.

## 왜 만들었나
"가고 싶은 곳은 있는데 뭘 어떻게 해야 할지 몰라" 멈추는 순간을 없애기 위해.
회사 하나 넣으면 리서치부터 액션까지 쭉 끌어준다.

## 로드맵 (전체 비전)
- **1단계 (현재): 회사 타겟 모드** — 이 문서
- 2단계: 커리어 점검 모드 — 이력서 넣으면 시장 내 내 위치 + 월별 플랜
- 2단계: 경쟁력 리포트 — 반복 사용 데이터로 연차·시장 대비 경쟁력 객관화
