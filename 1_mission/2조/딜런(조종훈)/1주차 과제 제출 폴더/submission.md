# 1주차 — 나만의 OS 만들기

## 🎯 미션 1. 내 OS 재료 찾기

**과정 (어떻게 찾았나):**
에밀리 인터뷰 스킬로 "내 삶에 필요한 게 뭔지"를 팠다. 처음엔 커리어 방향 탐색, 여유 시간 활용 등 여러 후보를 돌았는데 계속 추상적으로 막혔다. 진짜 재료는 "뭘 만들지" 고민이 아니라 **"나는 이상하게 ___가 거슬려"**라는 질문에서 나왔다.

**결과:**
- 핵심 마찰: 여유 시간이 늘수록 퇴근만 기다리는 시간이 괴로웠다. 노트에 "내 젊음이 아깝다"고 적었다.
- 그 밑의 진짜 감각: 시간이 *없어서*가 아니라 시간이 *내 것이 아니라서* 괴롭다 = **주도권의 부재.**
- 내가 원하는 한 조각: "회사 일과 관련 없이, 나를 위한 커피챗 하기"

**느낀 점:**
좋은 아이디어를 찾으려 할수록 막혔다. "남들은 별거 아닌데 나만 거슬리는 것"에서 나만의 시각이 나왔다. 나는 할 일 없는 시간을 '꿀'이 아니라 '고통'으로 느끼는데, 그게 "내가 먼저 주도한다"는 내 동력의 증거였다.

---

## 💡 미션 2. 내 OS 기획

**기획 내용:**
**커리어 OS — 회사 타겟 모드** — 가고 싶은 회사가 생겼을 때, **리서치 → 핏 점수 → 액션(커피챗 or 지원)**까지 한 흐름으로 끌어주는 OS.

- 표면 목적: 가고 싶은 회사가 생겼을 때 "뭘 어떻게 해야 할지 몰라" 멈추는 순간을 없애기
- 숨은 목적(B→A): 어디에 끌렸는지가 쌓여 내 방향이 드러난다
- 구조: ①회사 리서치(5앵글) ②JD 핏 스코어링 ③유사 기업 + 산업 트렌드 ④커피챗 or 지원 액션 분기

**티로(세션 기록) 활용:**
동기 세션 티로를 *차별점 기준*으로 활용했다. 해당 OS는 "생각을 쌓아 농익혀 콘텐츠로 자산화"하는 구조 — 모으고 기다리는 방향. 내 OS는 정반대로 "지금 밖으로 나가 사람을 만나게" 미는 구조다. 티로를 베끼는 대신, 내 것을 또렷하게 만드는 거울로 썼다.

**왜 이게 나인가:**
BD 경력의 최고 무기가 "연결·명분 생성"인데 그동안 전부 회사를 위해서만 썼다. 이 OS는 그 능력을 처음으로 나 자신에게 쓰게 한다. 나에게 에너지를 주는 구조.

**막혔던 점 / 어떻게 풀었나:**
- "이 OS 남도 쓰나?"에 자꾸 막혔다 → 1주차는 "나만의" OS가 정답. 가장 개인적인 게 가장 잘 굴러간다.
- 동기 발표와 비교하며 자꾸 작아 보였다 → 비교를 멈추고 "내 고통에서 나온 것"에 집중하니 풀렸다.
- 커피챗의 진짜 마찰 = "명분이 없어서" → BD가 회사 딜에 명분 만들던 능력을 나에게 쓰는 것으로 OS 정체를 확정.

## ⚙️ 미션 3. 내 OS 구현

**결과물:**
Next.js 웹앱 + Claude Code 스킬로 구현. 두 레이어:
- **웹앱** (배포 완료): 이력서 입력 → 회사 검색/링크 입력 → 리서치·핏점수·액션플랜 자동 생성
- **스킬파일** (`커피챗-os/` 폴더): `SKILL.md`(OS 본체)

**핵심 설계:**
- 이력서를 먼저 넣고 나서 회사를 검색하는 순서 (내가 누구인지 파악이 선행)
- 잘 모르는 회사는 채용 URL or 뉴스를 붙여넣으면 AI가 1차 소스로 사용 (할루시네이션 방지)
- 리서치 → JD 핏 점수 → 유사 기업 + 트렌드 → 커피챗 or 지원 액션까지 STEP별 카드로 출력
- 반복 사용 → `log.md`에 쌓여 커리어 방향 파악 (B→A 엔진)
