# 1주차 — 나만의 OS 만들기 🛠️

> 미션을 진행하며 **과정과 결과물**을 기록해주세요. (다 못 채워도 OK, 한 것 위주로!)

## 🎯 미션 1. 내 OS 재료 찾기
> 인터뷰 스킬(아이데이션)으로 "내 삶에 필요한 게 뭔지" 찾기
- **과정 (어떻게 찾았나):** 인터뷰 스킬로 하루 중 가장 시간이 많이 걸리는 일부터 시작해, 일과 삶 양쪽을 들여다봤다. "무슨 기능을 만들까"가 아니라 "어디가 자꾸 걸리나"부터 찾는 방식으로 진행.
- **결과:**

  🟡 **내 OS 재료 카드**

  | | |
  |---|---|
  | **영역** | 일 — 도그 피트니스 교육 영상 편집 |
  | **걸리는 지점** | 편집마다 준비 장면 지우고, 잘된 세트 골라내는 것 |
  | **지금은** | 캡컷·핸드폰 편집기로 직접 보면서 → 세트 선택 → 운동별로 자르기 → 저장 → 카카오채널톡으로 정리글과 함께 전송 |
  | **OS가 된다면** | 영상 찍으면 → 좋은 세트가 골라지고 → 운동별 파일로 자동 분리·저장 |
  | **한 문장** | "보호자와의 약속을 일찍 마무리하는 홀가분한 맘으로 내가 하고 싶은 걸 더 잘할 것 같아" |
  | **첫 한 걸음** | 영상을 넣으면 운동별로 구간을 나눠주는 것 하나부터 |

- **느낀 점:** 모든 답은 내가 가지고 있죠. 질문에 답하면서 생각과 과정이 조금씩 보이는 것 같아요.

## 🧩 미션 2. 내 OS 기획
> 인터뷰 결과 + 세션 내용(흐민·배짱·키노) 활용해 기획
- **기획 내용:**

  **도그 피트니스 교육 영상 편집 OS — 3단계 설계**

  **1단계 (이번 주 목표):** 액션캠으로 찍은 영상을 컴퓨터에서 열고, 운동별 구간의 시작·끝 시간을 텍스트로 적어두면 → 스크립트가 자동으로 운동별 파일로 잘라서 저장해주는 도구. 지금 캡컷에서 일일이 자르고 이름 붙이고 저장하는 과정을 없앤다.

  **2단계:** 같은 운동 여러 세트 중에서 잘된 세트를 AI가 후보로 추려주기.

  **3단계:** 보호자에게 보낼 교육 정리글 초안까지 자동 생성. 찍고 나면 편집·정리·전송 준비가 한 흐름으로.

- **막혔던 점 / 어떻게 풀었나:** 처음엔 영상 전체를 AI가 분석해서 운동 구간을 자동으로 찾아주길 원했는데, 같은 장소에서 계속 찍는 영상이라 장면 전환 감지가 안 됐다. → 촬영할 때 초록(시작)/주황(끝) 카드를 들어 마커를 만들고, 그 색상을 감지해서 자르는 방식으로 해결.

## ⚙️ 미션 3. 내 OS 구현
> 실제로 만들어본 것
- **결과물:** 도그 피트니스 교육 영상 자동 분할 도구 3종

  **① 영상자르기.py** — 타임스탬프 직접 입력해서 자르기 (기본형)

  **② 자동분석자르기.py** — 모션 감지로 자동 구간 탐지 (실험형)

  **③ 마커감지자르기.py** — 초록/주황 카드 마커 감지로 자동 분할 (완성형)
  - 촬영 시 운동 시작에 초록 카드, 끝에 주황 카드를 1~2초 들기
  - 스크립트가 색상 마커를 자동 감지 → 마커 사이 구간만 저장
  - 준비 과정·세팅 시간은 자동 제외
  - 10초 미만 오탐지 자동 필터링
  - 실제 54분 영상 테스트: 8개 운동 구간 자동 분리 성공
